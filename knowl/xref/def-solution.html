Definition 1.3.1. Solution to an Equation.

A solution to an equation in the variables \(x\) and \(y\) is an ordered pair \((a,b)\) that make the equation true when
  1. the first coordinate, \(a\text{,}\) is substituted into the equation for \(x\text{,}\) and
  2. the second coordinate, \(b\text{,}\) is substituted into the equation for \(y\text{.}\)
