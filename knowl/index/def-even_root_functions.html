Definition 4.4.9. Even Root Function.

Assume \(n\) is an even, positive integer and \(f \colon [0,\infty) \to \R\) is the function defined by \(f(x) = x^n\text{.}\) The \(n^\text{th}\) root function \(\sqrt[n]{\phantom{x}} \colon \R \to [0,\infty)\) given by \(\sqrt[n]{x} = x^{1/n}\) is defined to be the inverse of \(f\text{.}\)
