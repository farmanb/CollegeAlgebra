Definition 3.2.1. Linear Function.

A function \(f \colon \R \to \R\) is linear if the The Average Rate of Change of \(f\) is constant. That is, there exists a number \(m\) such that for all numbers \(a\) and \(b\)
\begin{equation*} m = \frac{f(b) - f(a)}{b - a}. \end{equation*}
We call the number \(m\) the slope of \(f\text{.}\) The number \(b\) is the \(y\)-coordinate of the \(y\)-intercept \(\left(0, b\right)\text{.}\)
