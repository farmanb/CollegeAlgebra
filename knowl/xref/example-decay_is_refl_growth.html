Example 5.1.11.

The exponential decay function \(\Exp{\frac{1}{2}}(x) = (1/2)^x\) is the reflection of the exponential growth function \(\Exp{2}(x) = 2^x\) because
\begin{equation*} \Exp{2}(-x) = \left(2\right)^{-x} = \left(\left(2\right)^{-1}\right)^x = \left(\frac{1}{2}\right)^x = \Exp{\frac{1}{2}}(x). \end{equation*}
