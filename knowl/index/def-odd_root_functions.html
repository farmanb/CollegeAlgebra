Definition 4.4.8. Odd Root Function.

Assume \(n\) is an odd, positive integer and \(f \colon \R \to \R\) is the function defined by \(f(x) = x^n\text{.}\) The \(n^\text{th}\) root function \(\sqrt[n]{\phantom{x}} \colon \R \to \R\) given by \(\sqrt[n]{x} = x^{1/n}\) is defined to be the inverse of \(f\text{.}\)
