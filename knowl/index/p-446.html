These identity elements are important because they provide a method for "undoing" the operation in the following sense. For any real number \(x\text{,}\) the additive inverse is the real number \(-x\) that satisfies
\begin{equation*} x + (-x) = 0\quad\text{and}\quad -x + x = 0. \end{equation*}
Similarly, for any non-zero real number \(x\text{,}\) the multiplicative inverse is the real number \(1/x\) that satisfies
\begin{equation*} x \times \frac{1}{x} = 1\quad\text{and}\quad \frac{1}{x} \times x = 1. \end{equation*}
