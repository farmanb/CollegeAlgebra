Definition 1.1.24. Closed Interval.

Let \(a\) and \(b\) be real numbers and assume that \(a \lt b\text{.}\) A closed interval is a set that takes one of the following forms
\begin{align*} [a,b] \amp= \left\{x \in \R \;\middle\vert\; a \leq x \leq b\right\},\\ [a, \infty) \amp = \left\{x \in \R ;\middle\vert\; a \leq x\right\},\, \text{or}\\ (-\infty, b] \amp = \left\{x \in \R \;\middle\vert\; x \leq b\right\}.\\ (-\infty, \infty) \amp = \R. \end{align*}
