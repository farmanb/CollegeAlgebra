    Section 5.4 Composition Inverses

    In keeping with the analogy with operations on numbers, recall the operations of addition and multiplication both have distinguished elements. The number \(0\) is called the additive identity because every real number \(x\) remains unchanged when \(0\) is added to it
    \begin{equation*} x + 0 = x \quad\text{and}\quad 0 + x = x. \end{equation*}
    Similarly, the number \(1\) is called the multiplicative identity because every real number \(x\) remains unchanged when it is multiplied by \(1\)
    \begin{equation*} x \times 1 = x \quad\text{and}\quad 1 \times x = x. \end{equation*}

    Subsection 5.4.1 Identities

    We introduced the idea of an identity using two familiar operations to highlight that the identity element depends on the operation. However, the essential behavior of the identity element is the same: when we perform the operation using the identity, nothing changes. This tells us exactly how the identity element for the function composition operation should behave.

    Definition 5.4.1. Identity Function.

    Assume \(A\) is a set. The identity function on \(A\) is the function \(\id \colon A \to A\) defined by
    \begin{equation*} \id_A(x) = x. \end{equation*}
    As we have already seen, the order of function composition matters. Complicating matters more, not all functions can be composed. For a function \(f \colon A \to B\text{,}\)
    • the composition \(g \circ f\) only makes sense when the domain of \(g\) contains the range of \(f\text{,}\) and
    • the composition \(f \circ g\) only makes sense when the range of \(g\) is contained in the domain of \(f\text{.}\)
    For this reason, there is not one single identity function that satisfies the condition \(f \circ \id = f\) and \(\id \circ f = f\text{.}\)
    Instead, we need to consider the identity functions on the domain and codomain of \(f\text{.}\) The identity function on \(A\) satisfies \(f \circ \id_A = f\) because
    \begin{equation*} (f \circ \id_A)(x) = f\left(\id_A(x)\right) = f(x) \end{equation*}
    and the identity function on \(B\) satisfies \(\id_B \circ f = f\) because
    \begin{equation*} (\id_B \circ f)(x) = \id_B\left(f(x)\right) = f(x)\text{.} \end{equation*}

    Subsection 5.4.2 Composition Inverses

    The identity elements for these operations are important because they provide a method for "undoing" the operation. This is easiest to understand by considering addition and multiplication first. For any real number \(c\text{,}\) the additive inverse is the real number \(-c\) that satisfies
    \begin{equation*} c + (-c) = 0\quad\text{and}\quad -c + c = 0. \end{equation*}
    If we add the number \(c\) to another number \(x\text{,}\) then we obtain the number \(x + c\text{.}\) We can "undo" this operation by performing the operation "add the additive inverse of \(c\)" -- which we normally just call "subtract \(c\)" -- in the sense that
    \begin{equation} (x + c) + (-c) = x + (c - c) = x + 0 = x.\tag{5.4.1} \end{equation}
    Similarly, we note that the operation "add c" will "undo" the operation "subtract \(c\)" in the sense that
    \begin{equation} (x - c) + c = x + (-c + c) = x + 0 = x.\tag{5.4.2} \end{equation}
    The operations "add \(c\)" and "subtract \(c\)" define functions \(\operatorname{add}_c \colon \R \to \R\) and \(\operatorname{add}_{-c} \colon \R \to \R\) by
    \begin{equation*} \operatorname{add}_c(x) = x + c \quad\text{and}\quad \operatorname{add}_{-c}(x) = x - c. \end{equation*}
    We can view (5.4.1) and (5.4.2) in terms of composition:
    \begin{align*} \operatorname{add}_{-c} \circ \operatorname{add}_c(x) \amp= \operatorname{add}_{-c}(x + c)\\ \amp= (x + c) - c \\ \amp= x \end{align*}
    \begin{align*} \operatorname{add}_c \circ \operatorname{add}_{-c}(x) \amp= \operatorname{add}_c(x - c)\\ \amp= (x - c) + c \\ \amp= x. \end{align*}
    The story is similar for multiplication. For any non-zero real number \(c\text{,}\) the multiplicative inverse of \(c\) is the real number \(c^{-1} = 1/c\) that satisfies
    \begin{equation*} c \times \frac{1}{c} = 1\quad\text{and}\quad \frac{1}{c} \times c = 1. \end{equation*}
    Multiplication by \(c\) and multiplication by \(a^{-1}\) define functions \(\operatorname{mul}_c, \operatorname{mul}_{c^{-1}} \colon \R \to \R\) by
    \begin{equation*} \operatorname{mul}_c(x) = cx \quad\text{and}\quad \operatorname{mul}_{c^{-1}}(x) = \frac{1}{c}x. \end{equation*}
    Just as with addition, these two functions "undo" one another in the sense that
    \begin{align*} \operatorname{mul}_c \circ \operatorname{mul}_{c^{-1}}(x) \amp= \operatorname{mul}_c\left(c^{-1}x\right)\\ \amp= c\left(\frac{1}{c}x\right) \\ \amp= x \end{align*}
    \begin{align*} \operatorname{mul}_{c^{-1}} \circ \operatorname{mul}_c(x) \amp= \operatorname{mul}_{c^{-1}}(cx)\\ \amp= \frac{1}{c}(cx)\\ \amp= x. \end{align*}
    Less formally, one often says "division by \(c\)" instead of "multiplication by \(c^{-1}\text{.}\)"
    These are two canonical examples of functions that are invertible, and the ideas captured in inverting addition and multiplication motivate the more general definition.

    Definition 5.4.2. Composition Inverse.

    Assume \(f \colon A \to B\) is a function. The composition inverse, if it exists, is a function \(f^{-1} \colon B \to A\) that satisfies
    \begin{equation*} f^{-1} \circ f = \id_A \quad\text{and}\quad f \circ f^{-1} = \id_B \end{equation*}
    or, equivalently, for every \(x \in A\text{,}\) \(f^{-1} \circ f(x) = x\) and for every \(y \in B\text{,}\) \(f \circ f^{-1}(y) = y\text{.}\)
    If \(f\) has a composition inverse, then we say \(f\) is invertible.

    Warning 5.4.3.

    Mathematicians commonly use the word "inverse" or "invertible" without explicitly stating the operation. It is important to infer the meaning from the context. If the object being discussed is a function, then "inverse" means composition inverse, not multiplicative inverse. That is to say,
    \begin{equation*} f^{-1}(x) \neq \frac{1}{f(x)} = f(x)^{-1}. \end{equation*}
    In order to avoid confusion, we will avoid using the notation \(f(x)^{-1}\) and write either \(1/f(x)\) or
    \begin{equation*} \frac{1}{f(x)} \end{equation*}
    whenever we refer to a fraction with the function \(f(x)\) in the denominator.

    Subsection 5.4.3 Properties of Invertible Functions

    For a function \(f \colon A \to B\) with a composition inverse, the codomain of \(f\) and the domain of \(f^{-1}\) agree. For every \(y \in B\text{,}\) the element \(x = f^{-1}(y) \in A\) satisfies
    \begin{equation*} y = f \circ f^{-1}(y) = f(x) \end{equation*}
    by definition. This provides useful information about the range of an invertible function \(f\text{.}\)

    Theorem 5.4.4.

    If \(f \colon A \to B\) is a function with a composition inverse, then the range of \(f\) is \(f(A) = B.\)
    The definition of a composition inverse is symmetric in the sense that the inverse of \(f\) is \(f^{-1}\) and the inverse of \(f^{-1}\) is \(f\text{.}\)

    Theorem 5.4.5.

    If \(f \colon A \to B\) is a function with inverse \(f^{-1} \colon B \to A\text{,}\) then \(f^{-1}\) has a composition inverse and
    \begin{equation*} \left(f^{-1}\right)^{-1} = f. \end{equation*}

    Example 5.4.6.

    In the language of Definition 5.4.2, the functions "add \(c\)" and "add \(-c\)" (or "subtract \(c\)") are a pair of inverse functions
    \begin{equation*} \left(\operatorname{add}_{c}\right)^{-1} = \operatorname{add}_{-c} \quad\text{and}\quad \left(\operatorname{add}_{-c}\right)^{-1} = \operatorname{add}_c. \end{equation*}
    Similarly, the functions "multiply by \(c\)" and "multiply by \(c^{-1}\)" (or "divide by \(c\)") are a pair of inverse functions
    \begin{equation*} \left(\operatorname{mul}_c\right)^{-1} = \operatorname{mul}_{c^{-1}} \quad\text{and}\quad \left(\operatorname{mul}_{c^{-1}}\right)^{-1} = \operatorname{mul}_c. \end{equation*}

    Example 5.4.7.

    In general, a linear function \(f(x) = mx + b\) is invertible if \(m \neq 0\text{.}\) To evaluate \(f\) at the real number \(x\text{,}\) we perform the following operations
    1. Multiply \(x\) by \(m\text{,}\)
    2. Add \(b\) to \(mx\text{.}\)
    This allows us to decompose \(f\) as the function "add \(b\)" composed with the function "multiply by \(m\)"
    \begin{equation*} f(x) = \operatorname{add}_b \circ \operatorname{mul}_m(x). \end{equation*}
    Since each function is invertible, we can invert \(f\) by performing the inverse operations in reverse order, like peeling back the layers of an onion.
    Write \(y = mx + b\) for the result of applying \(f\) to \(x\text{.}\) We would invert (or "undo") the operation of applying \(f\) to \(x\) by performing the operations
    1. Subtract \(b\) from \(y\)
    2. Divide \(y - b\) by \(m\) (Important: This is why we required \(m \neq 0\text{!}\))
    Note this is equivalent to solving the equation \(y = mx + b\) for \(x\) in terms of \(y\)
    In the language of Example 5.4.6, the inverse of \(f\) is the composition
    \begin{align*} f^{-1}(y) \amp= \left(\operatorname{mul}_m\right)^{-1} \circ \left(\operatorname{add}_{b}\right)^{-1}(y)\\ \amp=\operatorname{mul}_{m^{-1}} \circ \operatorname{add}_{-b}(y) \\ \amp= \operatorname{mul}_{m^{-1}}(y - b)\\ \amp=\frac{y - b}{m}\\ \amp=\frac{1}{m}y - \frac{b}{m}. \end{align*}
    Finally, we can check these are in fact inverses of one another by checking the two compositions explicitly:
    \begin{align*} f \circ f^{-1}(y) \amp= f\left(\frac{1}{m}y - \frac{b}{m}\right)\\ \amp= m\left(\frac{1}{m}y - \frac{b}{m}\right) + b\\ \amp= \frac{m}{m} y - \frac{mb}{m} + b\\ \amp= y - b + b\\ \amp=y \end{align*}
    \begin{align*} f^{-1} \circ f(x) \amp = f^{-1}(mx + b)\\ \amp= \frac{1}{m}(mx + b) - \frac{b}{m}\\ \amp= \frac{m}{m} x + \frac{b}{m} - \frac{b}{m}\\ \amp= x. \end{align*}
    Our observations about compositions of invertible functions in Example 5.4.7 are true in general.

    Formula 5.4.8.

    Assume \(f\) and \(g\) are invertible functions. If the composition is well-defined, then \(g \circ f\) is invertible and
    \begin{equation*} \left(g \circ f\right)^{-1} = f^{-1} \circ g^{-1}. \end{equation*}
    As a final example, we illustrate why it is necessary to pay close attention to the domain of a function.

    Example 5.4.9.

    Consider the quadratic function \(f \colon \R \to [0,\infty)\) defined by \(f(x) = x^2\text{.}\) This function is not invertible .
    To see why, we must observe that if an inverse function exists, then by Definition 5.4.2, it must have the form \(f^{-1} \colon [0,\infty) \to \R\) and for every positive number \(y\text{,}\) we must assign a single number \(x = f^{-1}(y)\) such that
    \begin{equation*} x^2 = f(x) = f \circ f^{-1}(y) = y. \end{equation*}
    However, there are always two such values: a positive number \(x\) and its negative \(-x\text{.}\) This means we must choose one of these two values to be \(f^{-1}(y)\text{.}\) Unfortunately, both decisions fail to produce an inverse for \(f\text{.}\)
    If we decide that \(x = f^{-1}(y)\) is the positive number such that \(x^2 = y\text{,}\) then
    \begin{align*} f^{-1} \circ f(-x) \amp= f^{-1}\left(\left(-x\right)^2\right)\\ \amp= f^{-1}(x^2)\\ \amp= f^{-1}(y)\\ \amp= x \neq -x. \end{align*}
    On the other hand, if we decide that \(-x = f^{-1}(y)\) is the negative number such that \((-x)^2 = x^2 = y\text{,}\) then
    \begin{equation*} f^{-1} \circ f(x) = f^{-1}(x^2) = -x \neq x. \end{equation*}
    Fortunately, these two problems suggest simple solutions.
    In the first case, we simply exclude the negative numbers from the domain of \(f\text{.}\) This operation is called restriction and produces a new function \(g \colon [0,\infty) \to [0,\infty)\) defined by \(g(x) = x^2\text{.}\) This function becomes invertible and its inverse is the principle square root function \(g^{-1} \colon [0,\infty) \to [0,\infty)\) defined by \(g^{-1}(y) = \sqrt{y}\text{.}\) For every non-negative number \(x\text{,}\)
    \begin{align*} g^{-1} \circ g(x) \amp= g^{-1}(x^2)\\ \amp= \sqrt{x^2}\\ \amp= \abs{x}\\ \amp= x, \end{align*}
    \begin{align*} g \circ g^{-1}(x) \amp= g(\sqrt{x})\\ \amp= \sqrt{x}^2\\ \amp= x. \end{align*}
    In the second case, we restrict the domain of \(f\) to the negative numbers. This produces a new function \(h \colon (-\infty,0] \to [0,\infty)\) defined by \(h(x) = x^2\text{.}\) The function \(h\) is invertible and its inverse is the function \(h^{-1} \colon [0,\infty) \to (-\infty,0]\) defined by \(h^{-1}(y) = -\sqrt{y}\text{.}\) For every non-negative real number \(x\text{,}\)
    \begin{align*} h^{-1} \circ h(-x) \amp = h^{-1}\left(\left(-x\right)^2\right)\\ \amp= h^{-1}(x^2)\\ \amp= -\sqrt{x^2}\\ \amp= -\abs{x}\\ \amp= -x \end{align*}
    \begin{align*} h \circ h^{-1}(x) \amp= h\left(-\sqrt{x}\right)\\ \amp=\left(-\sqrt{x}\right)^2\\ \amp=(-1)^2\sqrt{x}^2\\ \amp=\sqrt{x}^2\\ \amp=x. \end{align*}
