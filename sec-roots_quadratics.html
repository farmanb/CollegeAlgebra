    Section 3.7 Roots of Quadratic Functions

    Subsection 3.7.1 The Quadratic Formula

    In order to accurately graph a quadratic function \(y = f(x) = ax^2 + bx + c\text{,}\) it is necessary to identify the \(x\)-intercepts for the function. A point \((x,0)\) in the Cartesian plane is an \(x\)-intercept for this function if
    \begin{equation*} 0 = ax^2 + bx + c. \end{equation*}
    This reduces the geometric problem of finding the \(x\)-intercepts of a quadratic to the algebraic problem finding solutions to equations.

    Definition 3.7.1.

    An equation of the form \(ax^2 + bx + c = 0\) is called a quadratic equation
    To find the solutions to the quadratic equation \(ax^2 + bx + c\text{,}\) we use The Vertex Form of a Quadratic to write
    \begin{equation*} 0 = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} \end{equation*}
    and solve this equation for \(x\)
    \begin{align*} \amp a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} = 0 \amp\amp \text{Add}\ \frac{b^2 - 4ac}{4a}\ \text{to both sides}\\ \amp a\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a} \amp\amp \text{Divide both sides by}\ a\\ \amp \left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2} \amp\amp \text{Take the square root of both sides}\\ \amp x + \frac{b}{2a} = \pm\sqrt{\frac{b^2 - 4ac}{4a^2}} \amp\amp \text{Simplify}\\ \amp x + \frac{b}{2a} = \pm\frac{\sqrt{b^2 - 4ac}}{\sqrt{4a^2}} \amp\amp \text{Simplify}\\ \amp x + \frac{b}{2a} = \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Subtract}\ \frac{b}{2a}\ \text{from both sides}\\ \amp x = - \frac{b}{2a} + \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify}\\ \amp x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify} \end{align*}
    It is important to note that
    \begin{equation*} x = \frac{-b + \sqrt{b^2 - 4ac}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{b^2 - 4ac}}{2a} \end{equation*}
    are real numbers only when \(b^2 - 4ac\) is a non-negative number. The number \(b^2 - 4ac\) is an important invariant of a quadratic function.

    Definition 3.7.2. The Discriminant.

    The discriminant of the quadratic function \(f(x) = ax^2 + bx + c\) is the number
    \begin{equation*} D = b^2 - 4ac \end{equation*}
    The discriminant provides both algebraic and geometric information about quadratic functions. On the algebraic side, the discriminant tells us how many solutions there are to the quadratic equation \(ax^2 + bx +c = 0\text{.}\)

    Formula 3.7.3. The Quadratic Formula.

    The quadratic equation
    \begin{equation*} ax + bx + c = 0 \end{equation*}
    has either two, one, or zero real solutions, depending on The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then the two solutions are
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{D}}{2a}. \end{equation*}
    2. If \(D = 0\text{,}\) then the only solution is
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} = \frac{-b}{2a}. \end{equation*}
    3. If \(D \lt 0\text{,}\) then there are no real solutions.
    If we pair The Quadratic Formula with Theorem 3.2.6, then we have an algorithmic way to factor any quadratic function.

    Theorem 3.7.4.

    The quadratic function \(f(x) = ax^2 + bx + c\) factors as
    \begin{equation*} f(x) = a\left(x - \frac{-b + \sqrt{D}}{2a}\right)\left(x - \frac{-b - \sqrt{D}}{2a}\right), \end{equation*}
    where \(D = b^2 - 4ac\) is the The Discriminant.

    Example 3.7.5.

    Factor the quadratic \(f(x) = 10x^2 - 11x + 3\text{.}\)
    Solution.
    The discriminant is
    \begin{equation*} D = (-11)^2 - 4(10)(3) = 121 - 120 = 1 \end{equation*}
    so the roots are
    \begin{align*} x \amp= \frac{-(-11) + \sqrt{1}}{2(10)} = \frac{11 + 1}{20} = \frac{12}{20} = \frac{3}{5},\,\text{and}\\ x \amp = \frac{-(-11) - \sqrt{1}}{2(10)} = \frac{11 - 1}{20} = \frac{10}{20} = \frac{1}{2}. \end{align*}
    By Theorem 3.7.4, the factorization is
    \begin{equation*} 10\left(x - \frac{1}{2}\right)\left(x - \frac{3}{5}\right) = 2\left(x - \frac{1}{2}\right)5\left(x - \frac{3}{5}\right) = (2x - 1)(5x - 3). \end{equation*}
    We can check our answer by using FOIL to see
    \begin{equation*} (2x - 1)(5x - 3) = 10x^2 - 6x - 5x + 3 = 10x^2 -11x + 3 = f(x). \end{equation*}
    On the geometric side, the discriminant tells us how many \(x\)-intercepts the graph of \(y = ax^2 + bx + c\) has.

    Theorem 3.7.6. The \(x\)-intercepts of a Quadratic Function.

    The quadratic function \(y = ax^2 + bx + c\) has either two, one, or zero \(x\)-intercepts, depending on the The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then there are two \(x\)-intercepts,
      \begin{equation*} \left(\frac{-b + \sqrt{D}}{2a},0\right) \quad\text{and}\quad \left(\frac{-b - \sqrt{D}}{2a},\right). \end{equation*}
    2. If \(D = 0\text{,}\) then there is one \(x\)-intercept
      \begin{equation*} \left(\frac{-b}{2a},0\right). \end{equation*}
      Note this is the vertex.
    3. If \(D \lt 0\text{,}\) then there are no \(x\)-intercepts.

    Example 3.7.7.

    Find the \(x\)-intercepts of the function \(f(x) = 3x^2 + 2x + 4\text{.}\)
    Solution.
    Since the discriminant is
    \begin{equation*} D = 2^2 - 4(3)(4) = 4 - 48 = -44 \lt 0, \end{equation*}
    this function does not have any \(x\)-intercepts.

    Example 3.7.8.

    Find the \(x\)-intercepts of the function \(f(x) = -3x^2 - 5x + 2\text{.}\)
    Solution.
    The discriminant of this quadratic is
    \begin{equation*} D = (-5)^2 - 4(-3)(2) = 25 + 24 = 49 = 7^2. \end{equation*}
    The solutions to the equation \(-3x^2 - 5x + 2 = 0\) are given by the formula
    \begin{equation*} x = \frac{-b \pm \sqrt{D}}{2a} = \frac{-(-5) \pm \sqrt{7^2}}{2(-3)} = \frac{5 \pm 7}{-6} \end{equation*}
    This tells us that one intercept occurs when
    \begin{equation*} x = \frac{5 + 7}{-6} = - \frac{12}{6} = -2 \end{equation*}
    and the other occurs when
    \begin{equation*} x = \frac{5 - 7}{-6} = \frac{-2}{-6} = \frac{1}{3}. \end{equation*}
    Therefore the \(x\)-intercepts are
    \begin{equation*} \left(-2,0\right) \quad\text{and}\quad \left(\frac{1}{3},0\right). \end{equation*}

    Subsection 3.7.2 Graphing Qudaratics

    We can simplify the process of graphing a quadratic function by using the tools we have developed in this section.

    Algorithm 3.7.9. Graphing a Quadratic Function.

    1. Use the The Vertex Formula to find the vertex, \((h,k)\text{,}\) of the parabola. Plot the vertex in the plane.
    2. Factor or use the The Quadratic Formula to find the \(x\)-intercept(s). If there are \(x\)-intercepts, plot them in the plane.
    3. Plot the \(y\)-intercept, \((0, f(0))\text{,}\) in the plane.
    4. Draw a parabola through the points in Steps (1) through (3).
      • The parabola will open upwards if \(0 \lt a\text{.}\)
      • The parabola will open downwards if \(a \lt 0\text{.}\)
