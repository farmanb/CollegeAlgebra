    Section 3.5 Quadratic Functions

    Objectives

    • Place a quadratic function in vertex form,
    • Find the vertex of a quadratic function,
    In this section, we study polynomial functions of degree 2. These polynomial functions are often called quadratic functions.

    Definition 3.5.1. Quadratic Function.

    A function \(f \colon \R \to \R\) is quadratic if it has the form
    \begin{equation*} f(x) = ax^2 + bx + c \quad\text{or}\quad y = ax^2 + bx + c, \end{equation*}
    where \(a\text{,}\) \(b\text{,}\) and \(c\) are numbers, and \(a \neq 0\text{.}\)

    Subsection 3.5.1 Vertex Form

    Just as with lines, quadratic functions are easier to understand through visualizations. The simplest quadratic function to understand is \(y = x^2\text{.}\) We call the the graph of \(y = x^2\) a parabola. The point \((0,0)\) is called the vertex.
    As we have already seen in Section 2.5, the graph of this function is symmetric about the \(y\)-axis, which is the vertical line \(x = 0\) through the vertex. While general quadratics are slightly more complicated, we can use the properties of \(y = x^2\) to graph the class of quadratics in the following form.

    Definition 3.5.2. The Vertex Form of a Quadratic.

    A quadratic function, \(f(x)\text{,}\) is in vertex form if there are numbers \(h\) and \(k\) such that
    \begin{equation*} f(x) = a(x - h)^2 + k. \end{equation*}
    Using the methods of Section 2.4, we can graph quadratics in this form by following the steps below.

    Algorithm 3.5.3. Graphing Quadratic Functions in Vertex Form.

    To graph of the quadratic function \(y = a(x - h)^2 + k\)
    1. Translate the graph of \(y = x^2\) horizontally by \(h\) units.
      • If \(0 \lt h\text{,}\) then the translation is to the right.
      • If \(h \lt 0\text{,}\) then the translation is to the left.
    2. Scale the graph of \((x - h)^2\) by \(\abs{a}\text{.}\)
      • If \(1 \lt a\text{,}\) then the graph is stretched.
      • If \(a \lt 1\text{,}\) then the graph is compressed.
    3. If \(a \lt 1\text{,}\) reflect the graph of \(\abs{a}(x - h)^2\) over the \(x\)-axis.
    4. Translate the graph of \(a(x - h)^2\) vertically by \(k\) units.
      • If \(k \lt 0\text{,}\) then the translation is down.
      • If \(0 \lt k\text{,}\) then the translation is up.

    Example 3.5.4.

    Graph the function \(f(x) = -2(x+1)^2 + 1\text{.}\)
    Solution.
    Following Algorithm 3.5.3, we perform the following operations on the graph of the function \(y = x^2\text{.}\)
    1. Translate the graph of \(y = x^2\) to the left by 1 to obtain the graph of \(y = (x + 1)^2\text{.}\)
    2. Stretch the graph of \(y = (x + 1)^2\) by a factor of 2 to obtain the graph of \(y = 2(x + 1)^2\text{.}\)
    3. Reflect the graph of \(y = 2(x + 1)^2\) across the \(x\)-axis to obtain the graph of \(y = -2(x + 1)^2\text{.}\)
    4. Translate the graph of \(y = -2(x + 1)^2\) up by 1 to obtain the graph of \(y = -2(x+1)^2 + 1\text{.}\)
    These operations are shown below.
    If we follow the vertex, we can see the first operation moves the point \((0,0)\) to \((h,0)\text{.}\) Since the point \((h,0)\) lies on the \(x\)-axis, the second and third operation do not move the point \((h,0)\text{.}\) The fourth operation moves the point \((h,0)\) to the point \((h,k)\text{.}\) Since these operations preserve the general shape of the graph, a quadratic function in the form
    \begin{equation*} y = a(x - h)^2 + k \end{equation*}
    is a parabola with vertex at \((h,k)\text{.}\) Hence the reason for the name vertex form.
    Similarly, if we follow the line of symmetry for \(y = x^2\text{,}\) then we can see the first operation moves the vertical line \(x = 0\) to the vertical line \(x = h\text{.}\) The following three operations do not change the vertical line, so the graph of \(y = a(x - h)^2 + k\) is symmetric about the vertical line \(x = h\text{.}\)
    Finally, we observe the third operation may change the direction in which the parabola opens.

    Definition 3.5.6.

    The graph of a parabola is
    • concave up if the parabola opens upwards like \(y=x^2\text{.}\)
    • concave down if the parabola opens downwards like \(y = -x^2\text{.}\)
    For quadratics in vertex form \(f(x) = a(x - h)^2 + k\text{,}\) we can see that \(f\) is concave up whenever \(0 \lt a\) and concave down whenever \(a \lt 0\text{.}\)

    Subsection 3.5.2 Completing the Square

    Surprisingly, every quadratic function can be placed into vertex form. Paired with the observations above, this tells us the graph of every quadratic function is a parabola. Placing a general quadratic function into vertex form will require the method of completing the square.
    Remember that when we multiply two binomials, we must distribute. We can remember how to distribute using the mnemonic FOIL: First, Outer, Inner, Last:
    If we use this method to multiply a binomial by itself, then we find the useful formula
    \begin{align*} (x + h)^2 \amp = (x + h)(x + h)\\ \amp = xx + xh + hx + hh\\ \amp = x^2 + hx + hx + h^2\\ \amp = x^2 + 2hx + h^2. \end{align*}
    We call a quadratic function of the form \(f(x) = x^2 + 2hx + h^2\) a perfect square.
    Given a quadratic equation \(f(x) = ax^2 + bx + c\text{,}\) we can use the formula
    \begin{equation*} (x + h)^2 = x^2 + 2hx + h^2 \end{equation*}
    to complete the square as follows.

    Algorithm 3.5.8. Completing the Square.

    To complete the square with the quadratic function \(f(x) = ax^2 + bx + c\)
    1. Factor an \(a\) from the first two terms to get
      \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x\right) + c. \end{equation*}
    2. Add and subtract \(\left(\frac{b}{2a}\right)^2\) inside the parentheses to get
      \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    3. Factor the first three terms inside the parentheses as a perfect square
      \begin{equation*} f(x) = a\left(\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    4. Distribute the \(a\) through the sum
      \begin{equation*} f(x) = a\left(x + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c. \end{equation*}
    The number \(h = -\frac{b}{2a}\) is the \(x\)-coordinate of the vertex, and the number
    \begin{align*} k \amp= f(h)\\ \amp = f\left(-\frac{b}{2a}\right)\\ \amp = a\left(-\frac{b}{2a} + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c\\ \amp =- a\left(\frac{b}{2a}\right)^2 + c\\ \amp= -a\left(\frac{b^2}{4a^2}\right) + c\\ \amp= -\frac{b^2}{4a} + \frac{4ac}{4a}\\ \amp = - \frac{b^2 - 4ac}{4a}. \end{align*}
    is the \(y\)-coordinate of the vertex.

    Formula 3.5.9. The Vertex Formula.

    Let \(f(x) = ax^2 + bx + c\) be a quadratic function. The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{b}{2a} \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{equation*} k = f(h) = f\left(-\frac{b}{2a}\right) = - \frac{b^2 - 4ac}{4a}. \end{equation*}
    As an ordered pair, the vertex of the graph of \(f(x)\) occurs at
    \begin{equation*} \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right) = \left(-\frac{b}{2a}, - \frac{b^2 - 4ac}{4a}\right). \end{equation*}

    Remark 3.5.10.

    Using The Vertex Formula, the vertex form of the quadratic function \(f(x) = ax^2 + bx + c\) is
    \begin{align*} a(x - h)^2 + k \amp= a\left(x - \left(-\frac{b}{2a}\right)\right)^2 + f\left(-\frac{b}{2a}\right)\\ \amp= a\left(x + \frac{b}{2a}\right)^2 + f\left(-\frac{b}{2a}\right) \end{align*}

    Example 3.5.11.

    Find the vertex form of the quadratic function \(f(x) = x^2 + 4x + 5\text{.}\)
    Solution.
    The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{4}{2(1)} = -\frac{4}{2} = -2 \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{equation*} k = f(-2) = (-2)^2 + 4(-2) + 5 = 4 - 8 + 5 = 1 \end{equation*}
    Therefore the vertex form of the quadratic function
    \begin{equation*} (x - (-2))^2 + 1 = (x + 2)^2 + 1. \end{equation*}

    Example 3.5.12.

    Find the vertex of the quadratic function \(f(x) = 3x^2 - 4x + 5\text{.}\)
    Solution.
    The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{(-4)}{2(3)} = \frac{4}{2(3)} = \frac{2}{3} \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{align*} k = f\left(\frac{2}{3}\right) \amp= 3\left(\frac{2}{3}\right)^2 - 4\left(\frac{2}{3}\right) + 5\\ \amp = 3\left(\frac{4}{9}\right) - \frac{8}{3} + 5\\ \amp = \frac{4}{3} - \frac{8}{3} + \frac{15}{3}\\ \amp = \frac{4 - 8 + 15}{3}\\ \amp = \frac{11}{3}. \end{align*}
    Therefore the vertex form of the quadratic function is
    \begin{equation*} f(x) = 3\left(x - \frac{2}{3}\right)^2 + \frac{11}{3}. \end{equation*}

    Subsection 3.5.3 The Quadratic Formula

    In order to accurately graph a quadratic function \(y = f(x) = ax^2 + bx + c\text{,}\) it is necessary to identify the \(x\)-intercepts for the function. A point \((x,0)\) in the Cartesian plane is an \(x\)-intercept for this function if
    \begin{equation*} 0 = ax^2 + bx + c. \end{equation*}
    This reduces the geometric problem of finding the \(x\)-intercepts of a quadratic to the algebraic problem finding solutions to equations.

    Definition 3.5.13.

    An equation of the form \(ax^2 + bx + c = 0\) is called a quadratic equation
    To find the solutions to the quadratic equation \(ax^2 + bx + c\text{,}\) we use The Vertex Form of a Quadratic to write
    \begin{equation*} 0 = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} \end{equation*}
    and solve this equation for \(x\)
    \begin{align*} \amp a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} = 0 \amp\amp \text{Add}\ \frac{b^2 - 4ac}{4a}\ \text{to both sides}\\ \amp a\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a} \amp\amp \text{Divide both sides by}\ a\\ \amp \left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2} \amp\amp \text{Take the square root of both sides}\\ \amp x + \frac{b}{2a} = \pm\sqrt{\frac{b^2 - 4ac}{4a^2}} \amp\amp \text{Simplify}\\ \amp x + \frac{b}{2a} = \pm\frac{\sqrt{b^2 - 4ac}}{\sqrt{4a^2}} \amp\amp \text{Simplify}\\ \amp x + \frac{b}{2a} = \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Subtract}\ \frac{b}{2a}\ \text{from both sides}\\ \amp x = - \frac{b}{2a} + \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify}\\ \amp x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify} \end{align*}
    It is important to note that
    \begin{equation*} x = \frac{-b + \sqrt{b^2 - 4ac}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{b^2 - 4ac}}{2a} \end{equation*}
    are real numbers only when \(b^2 - 4ac\) is a non-negative number. The number \(b^2 - 4ac\) is an important invariant of a quadratic function.

    Definition 3.5.14. The Discriminant.

    The discriminant of the quadratic function \(f(x) = ax^2 + bx + c\) is the number
    \begin{equation*} D = b^2 - 4ac \end{equation*}
    The discriminant provides both algebraic and geometric information about quadratic functions. On the algebraic side, the discriminant tells us how many solutions there are to the quadratic equation \(ax^2 + bx +c = 0\text{.}\)

    Formula 3.5.15. The Quadratic Formula.

    The quadratic equation
    \begin{equation*} ax + bx + c = 0 \end{equation*}
    has either two, one, or zero real solutions, depending on The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then the two solutions are
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{D}}{2a}. \end{equation*}
    2. If \(D = 0\text{,}\) then the only solution is
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} = \frac{-b}{2a}. \end{equation*}
    3. If \(D \lt 0\text{,}\) then there are no real solutions.
    If we pair The Quadratic Formula with Theorem 3.1.13, then we have an algorithmic way to factor any quadratic function.

    Theorem 3.5.16.

    The quadratic function \(f(x) = ax^2 + bx + c\) factors as
    \begin{equation*} f(x) = a\left(x - \frac{-b + \sqrt{D}}{2a}\right)\left(x - \frac{-b - \sqrt{D}}{2a}\right), \end{equation*}
    where \(D = b^2 - 4ac\) is the discriminant.

    Example 3.5.17.

    Factor the quadratic \(f(x) = 10x^2 - 11x + 3\text{.}\)
    Solution.
    The discriminant is
    \begin{equation*} D = (-11)^2 - 4(10)(3) = 121 - 120 = 1 \end{equation*}
    so the roots are
    \begin{align*} x \amp= \frac{-(-11) + \sqrt{1}}{2(10)} = \frac{11 + 1}{20} = \frac{12}{20} = \frac{3}{5},\,\text{and}\\ x \amp = \frac{-(-11) - \sqrt{1}}{2(10)} = \frac{11 - 1}{20} = \frac{10}{20} = \frac{1}{2}. \end{align*}
    By Theorem 3.5.16, the factorization is
    \begin{equation*} 10\left(x - \frac{1}{2}\right)\left(x - \frac{3}{5}\right) = 2\left(x - \frac{1}{2}\right)5\left(x - \frac{3}{5}\right) = (2x - 1)(5x - 3). \end{equation*}
    We can check our answer by using FOIL to see
    \begin{equation*} (2x - 1)(5x - 3) = 10x^2 - 6x - 5x + 3 = 10x^2 -11x + 3 = f(x). \end{equation*}
    On the geometric side, the discriminant tells us how many \(x\)-intercepts the graph of \(y = ax^2 + bx + c\) has.

    Theorem 3.5.18. The \(x\)-intercepts of a Quadratic Function.

    The quadratic function \(y = ax^2 + bx + c\) has either two, one, or zero \(x\)-intercepts, depending on the The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then there are two \(x\)-intercepts,
      \begin{equation*} \left(\frac{-b + \sqrt{D}}{2a},0\right) \quad\text{and}\quad \left(\frac{-b - \sqrt{D}}{2a},\right). \end{equation*}
    2. If \(D = 0\text{,}\) then there is one \(x\)-intercept
      \begin{equation*} \left(\frac{-b}{2a},0\right). \end{equation*}
      Note this is the vertex.
    3. If \(D \lt 0\text{,}\) then there are no \(x\)-intercepts.

    Example 3.5.19.

    Find the \(x\)-intercepts of the function \(f(x) = 3x^2 + 2x + 4\text{.}\)
    Solution.
    Since the discriminant is
    \begin{equation*} D = 2^2 - 4(3)(4) = 4 - 48 = -44 \lt 0, \end{equation*}
    this function does not have any \(x\)-intercepts.

    Example 3.5.20.

    Find the \(x\)-intercepts of the function \(f(x) = -3x^2 - 5x + 2\text{.}\)
    Solution.
    The discriminant of this quadratic is
    \begin{equation*} D = (-5)^2 - 4(-3)(2) = 25 + 24 = 49 = 7^2. \end{equation*}
    The solutions to the equation \(-3x^2 - 5x + 2 = 0\) are given by the formula
    \begin{equation*} x = \frac{-b \pm \sqrt{D}}{2a} = \frac{-(-5) \pm \sqrt{7^2}}{2(-3)} = \frac{5 \pm 7}{-6} \end{equation*}
    This tells us that one intercept occurs when
    \begin{equation*} x = \frac{5 + 7}{-6} = - \frac{12}{6} = -2 \end{equation*}
    and the other occurs when
    \begin{equation*} x = \frac{5 - 7}{-6} = \frac{-2}{-6} = \frac{1}{3}. \end{equation*}
    Therefore the \(x\)-intercepts are
    \begin{equation*} \left(-2,0\right) \quad\text{and}\quad \left(\frac{1}{3},0\right). \end{equation*}

    Subsection 3.5.4 Graphing Qudaratics

    We can simplify the process of graphing a quadratic function by using the tools we have developed in this section.

    Algorithm 3.5.21. Graphing a Quadratic Function.

    1. Use the The Vertex Formula to find the vertex, \((h,k)\text{,}\) of the parabola. Plot the vertex in the plane.
    2. Factor or use the The Quadratic Formula to find the \(x\)-intercept(s). If there are \(x\)-intercepts, plot them in the plane.
    3. Plot the \(y\)-intercept, \((0, f(0))\text{,}\) in the plane.
    4. Draw a parabola through the points in Steps (1) through (3).
      • The parabola will open upwards if \(0 \lt a\text{.}\)
      • The parabola will open downwards if \(a \lt 0\text{.}\)
