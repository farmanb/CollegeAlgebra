In keeping with the analogy with operations on numbers, recall the operations of addition and multiplication both have distinguished elements. The number \(0\) is the additive identity because every real number \(x\) remains unchanged when \(0\) is added to it
\begin{equation*} x + 0 = x \quad\text{and}\quad 0 + x = x. \end{equation*}
Similarly, the number \(1\) is the multiplicative identity because every real number \(x\) remains unchanged when it is multiplied by \(1\)
\begin{equation*} x \times 1 = x \quad\text{and}\quad 1 \times x = x. \end{equation*}
For function composition, there is an analogous identity element.
