If we use this method to multiply a binomial by itself, then we find the useful formula
\begin{align*} (x + h)^2 \amp = (x + h)(x + h)\\ \amp = xx + xh + hx + hh\\ \amp = x^2 + hx + hx + h^2\\ \amp = x^2 + 2hx + h^2. \end{align*}
We call a quadratic function of the form \(f(x) = x^2 + 2hx + h^2\) a perfect square.
