    Section 3.7 Quadratic Functions

    In this section, we study polynomial functions of degree 2. These polynomial functions are often called quadratic functions.

    Subsection 3.7.1 Vertex Form

    Definition 3.7.1. Quadratic Function.

    A function \(f \colon \R \to \R\) is quadratic if it has the form
    \begin{equation*} f(x) = ax^2 + bx + c \quad\text{or}\quad y = ax^2 + bx + c, \end{equation*}
    where \(a\text{,}\) \(b\text{,}\) and \(c\) are numbers, and \(a \neq 0\text{.}\)
    Just as with lines, quadratic functions are easier to understand through visualizations. The simplest quadratic function to understand is \(y = x^2\text{.}\) We call the graph of \(y = x^2\) a parabola. The point \((0,0)\) is called the vertex.
    As we have already seen in Section 2.4, the graph of this function is symmetric about the \(y\)-axis, which is the vertical line \(x = 0\) through the vertex. While general quadratics are slightly more complicated, we can use the properties of \(y = x^2\) to graph the class of quadratics in the following form.

    Definition 3.7.2. The Vertex Form of a Quadratic.

    A quadratic function, \(f(x)\text{,}\) is in vertex form if there are numbers \(h\) and \(k\) such that
    \begin{equation*} f(x) = a(x - h)^2 + k. \end{equation*}
    Using the methods of Section 2.3, we can graph quadratics in this form by following the steps below.

    Algorithm 3.7.3. Graphing Quadratic Functions in Vertex Form.

    To graph of the quadratic function \(y = a(x - h)^2 + k\)
    1. Translate the graph of \(y = x^2\) horizontally by \(h\) units.
      • If \(0 \lt h\text{,}\) then the translation is to the right.
      • If \(h \lt 0\text{,}\) then the translation is to the left.
    2. Scale the graph of \((x - h)^2\) by \(\abs{a}\text{.}\)
      • If \(1 \lt a\text{,}\) then the graph is stretched.
      • If \(a \lt 1\text{,}\) then the graph is compressed.
    3. If \(a \lt 1\text{,}\) reflect the graph of \(\abs{a}(x - h)^2\) over the \(x\)-axis.
    4. Translate the graph of \(a(x - h)^2\) vertically by \(k\) units.
      • If \(k \lt 0\text{,}\) then the translation is down.
      • If \(0 \lt k\text{,}\) then the translation is up.

    Example 3.7.4.

    Graph the function \(f(x) = -2(x+1)^2 + 1\text{.}\)
    Solution.
    Following Algorithm 3.7.3, we perform the following operations on the graph of the function \(y = x^2\text{.}\)
    1. Translate the graph of \(y = x^2\) to the left by 1 to obtain the graph of \(y = (x + 1)^2\text{.}\)
    2. Stretch the graph of \(y = (x + 1)^2\) by a factor of 2 to obtain the graph of \(y = 2(x + 1)^2\text{.}\)
    3. Reflect the graph of \(y = 2(x + 1)^2\) across the \(x\)-axis to obtain the graph of \(y = -2(x + 1)^2\text{.}\)
    4. Translate the graph of \(y = -2(x + 1)^2\) up by 1 to obtain the graph of \(y = -2(x+1)^2 + 1\text{.}\)
    These operations are shown below.
    If we follow the vertex, we can see the first operation moves the point \((0,0)\) to \((h,0)\text{.}\) Since the point \((h,0)\) lies on the \(x\)-axis, the second and third operation do not move the point \((h,0)\text{.}\) The fourth operation moves the point \((h,0)\) to the point \((h,k)\text{.}\) Since these operations preserve the general shape of the graph, a quadratic function in the form
    \begin{equation*} y = a(x - h)^2 + k \end{equation*}
    is a parabola with vertex at \((h,k)\text{.}\) Hence the reason for the name vertex form.
    Similarly, if we follow the line of symmetry for \(y = x^2\text{,}\) then we can see the first operation moves the vertical line \(x = 0\) to the vertical line \(x = h\text{.}\) The following three operations do not change the vertical line, so the graph of \(y = a(x - h)^2 + k\) is symmetric about the vertical line \(x = h\text{.}\)
    Finally, we observe the third operation may change the direction in which the parabola opens.

    Definition 3.7.6.

    The graph of a parabola is
    • concave up if the parabola opens upwards like \(y=x^2\text{.}\)
    • concave down if the parabola opens downwards like \(y = -x^2\text{.}\)
    For quadratics in vertex form \(f(x) = a(x - h)^2 + k\text{,}\) we can see that \(f\) is concave up whenever \(0 \lt a\) and concave down whenever \(a \lt 0\text{.}\)

    Subsection 3.7.2 Completing the Square

    Surprisingly, every quadratic function can be placed into vertex form. Paired with the observations above, this tells us the graph of every quadratic function is a parabola. Placing a general quadratic function into vertex form will require the method of completing the square.
    Remember that when we multiply two binomials, we must distribute. We can remember how to distribute using the mnemonic FOIL: First, Outer, Inner, Last:
    If we use this method to multiply a binomial by itself, then we find the useful formula
    \begin{align*} (x + h)^2 \amp = (x + h)(x + h)\\ \amp = xx + xh + hx + hh\\ \amp = x^2 + hx + hx + h^2\\ \amp = x^2 + 2hx + h^2. \end{align*}
    We call a quadratic function of the form \(f(x) = x^2 + 2hx + h^2\) a perfect square.
    Given a quadratic equation \(f(x) = ax^2 + bx + c\text{,}\) we can use the formula
    \begin{equation*} (x + h)^2 = x^2 + 2hx + h^2 \end{equation*}
    to complete the square as follows.

    Algorithm 3.7.8. Completing the Square.

    To complete the square with the quadratic function \(f(x) = ax^2 + bx + c\)
    1. Factor an \(a\) from the first two terms to get
      \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x\right) + c. \end{equation*}
    2. Add and subtract \(\left(\frac{b}{2a}\right)^2\) inside the parentheses to get
      \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    3. Factor the first three terms inside the parentheses as a perfect square
      \begin{equation*} f(x) = a\left(\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    4. Distribute the \(a\) through the sum
      \begin{equation*} f(x) = a\left(x + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c. \end{equation*}
    The number \(h = -\frac{b}{2a}\) is the \(x\)-coordinate of the vertex, and the number
    \begin{align*} k \amp= f(h)\\ \amp = f\left(-\frac{b}{2a}\right)\\ \amp = a\left(-\frac{b}{2a} + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c\\ \amp =- a\left(\frac{b}{2a}\right)^2 + c\\ \amp= -a\left(\frac{b^2}{4a^2}\right) + c\\ \amp= -\frac{b^2}{4a} + \frac{4ac}{4a}\\ \amp = - \frac{b^2 - 4ac}{4a}. \end{align*}
    is the \(y\)-coordinate of the vertex.

    Formula 3.7.9. The Vertex Formula.

    Let \(f(x) = ax^2 + bx + c\) be a quadratic function. The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{b}{2a} \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{equation*} k = f(h) = f\left(-\frac{b}{2a}\right) = - \frac{b^2 - 4ac}{4a}. \end{equation*}
    As an ordered pair, the vertex of the graph of \(f(x)\) occurs at
    \begin{equation*} \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right) = \left(-\frac{b}{2a}, - \frac{b^2 - 4ac}{4a}\right). \end{equation*}

    Remark 3.7.10.

    Using The Vertex Formula, the vertex form of the quadratic function \(f(x) = ax^2 + bx + c\) is
    \begin{align*} a(x - h)^2 + k \amp= a\left(x - \left(-\frac{b}{2a}\right)\right)^2 + f\left(-\frac{b}{2a}\right)\\ \amp= a\left(x + \frac{b}{2a}\right)^2 + f\left(-\frac{b}{2a}\right) \end{align*}

    Example 3.7.11.

    Find the vertex form of the quadratic function \(f(x) = x^2 + 4x + 5\text{.}\)
    Solution.
    The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{4}{2(1)} = -\frac{4}{2} = -2 \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{equation*} k = f(-2) = (-2)^2 + 4(-2) + 5 = 4 - 8 + 5 = 1 \end{equation*}
    Therefore the vertex form of the quadratic function
    \begin{equation*} (x - (-2))^2 + 1 = (x + 2)^2 + 1. \end{equation*}

    Example 3.7.12.

    Find the vertex of the quadratic function \(f(x) = 3x^2 - 4x + 5\text{.}\)
    Solution.
    The \(x\)-coordinate of the vertex is
    \begin{equation*} h = -\frac{(-4)}{2(3)} = \frac{4}{2(3)} = \frac{2}{3} \end{equation*}
    and the \(y\)-coordinate of the vertex is
    \begin{align*} k = f\left(\frac{2}{3}\right) \amp= 3\left(\frac{2}{3}\right)^2 - 4\left(\frac{2}{3}\right) + 5\\ \amp = 3\left(\frac{4}{9}\right) - \frac{8}{3} + 5\\ \amp = \frac{4}{3} - \frac{8}{3} + \frac{15}{3}\\ \amp = \frac{4 - 8 + 15}{3}\\ \amp = \frac{11}{3}. \end{align*}
    Therefore the vertex form of the quadratic function is
    \begin{equation*} f(x) = 3\left(x - \frac{2}{3}\right)^2 + \frac{11}{3}. \end{equation*}

    Subsection 3.7.3 Applications to Circles

    In Section 1.6 we only encountered circles in standard form. However, circles will not always be presented this way. We study how the The Vertex Formula can help us to identify the equation of a circle, even if it is not presented in standard form.
    Consider the equation \(x^2 - 4x + y^2 + 6x = -9.\) We can view \(f(x) = x^2 - 4x\) and \(g(y) = y^2 + 6x\) as quadratic functions in the variables \(x\) and \(y\text{,}\) respectively. The first coordinate of the vertex of \(f\) is
    \begin{equation*} h = -\frac{-4}{2} = \frac{4}{2} = 2 \end{equation*}
    and the second coordinate of the vertex as
    \begin{equation*} k = f(2) = 2^2 - 4(2) = 4 - 8 = -4\text{.} \end{equation*}
    This allows us to write
    \begin{equation*} f(x) = x^2 - 4x = (x - 2)^2 - 4. \end{equation*}
    Similarly, the first coordinate for the vertex of \(g\) is
    \begin{equation*} h = -\frac{6}{2(1)} = -3, \end{equation*}
    the second coordinate of the vertex of \(y^2 + 6x\) is
    \begin{equation*} k = g(-3) = (-3)^2 + 6(-3) = 9 - 18 = -9, \end{equation*}
    and so
    \begin{equation*} g(y) = (x - (-3))^2 - 9 = (x + 3)^2 - 9. \end{equation*}
    These two observations allow us to rewrite the original equation
    \begin{align*} x^2 - 4x + y^2 + 6x = -9 \amp\amp \text{Replace with vertex forms}\\ (x - 2)^2 - 4 + (y + 3)^2 - 9 = -9 \amp\amp \text{Add}\ 9\ \text{to both sides}\\ (x - 2)^2 - 4 + (y + 3)^2 = -9 + 9 \amp\amp \text{Simplify}\\ (x - 2)^2 - 4 + (y + 3)^2 = 0 \amp\amp \text{Add 4 to both sides}\\ (x - 2)^2 + (y + 3)^2 =4 \end{align*}
    Therefore the equation \(x^2 - 4x + y^2 + 6x = -9\) represents the circle of radius \(2\) centered at the point \((2,-3)\text{.}\)
