Definition 2.5.2. Solution to an Inequality.

A solution to an inequality in the variables \(x\) and \(y\) is an ordered pair \((a,b)\) that make the equation true when
  1. the first coordinate, \(a\text{,}\) is substituted into the inequality for \(x\text{,}\) and
  2. the second coordinate, \(b\text{,}\) is substituted into the inequality for \(y\text{.}\)
