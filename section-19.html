    Section 4.4 Graphing Inverse Functions

    Subsection 4.4.1 A Little Symmetry

    Assume \(A, B \subseteq \R\) and \(f \colon A \to B\) is a function with inverse \(f^{-1} \colon B \to A\text{.}\) In order to visualize the inverse of a function, it is helpful to recall from Definition 2.1.8 that the graph of \(f\) is the set of points
    \begin{equation*} \left\{\left(x, f(x)\right) \;\middle\vert\; x \in A\right\} \end{equation*}
    in the Cartesian plane.
    Since we have been conditioned by training to think about the input to a function being named \(x\) and the output of a function being named \(y\text{,}\) we opt to write \(f^{-1}(y)\) instead of \(f^{-1}(x)\text{.}\) This distinction is meaningless in practice, but prompts us to think about the input to \(f^{-1}\) as the output of \(f\text{.}\)
    We made these observations more precise in Theorem 4.3.10, which asserts that every \(y \in B\) can be written in the form \(y = f(x)\) for some \(x \in A\text{.}\) This allows us to describe the graph of \(f^{-1}\) as the set of points
    \begin{align*} \left\{\left(y, f^{-1}(y)\right) \;\middle\vert\; y \in B\right\} \amp= \left\{\left(f(x), f^{-1}\left(f(x)\right)\right) \;\middle\vert\; x \in A\right\}\\ \amp=\left\{\left(f(x), f^{-1} \circ f(x)\right) \;\middle\vert\; x \in A\right\}\\ \amp= \left\{\left(f(x), x\right) \;\middle\vert\; x \in A\right\}. \end{align*}
    Note that we started by describing the points on the graph of \(f^{-1}\) in terms of \(B\) and ended by describing the points on the graph of \(f^{-1}\) in terms of \(A\text{.}\) This description tells us that we can obtain the graph of \(f^{-1}\) from the graph of \(f\) by interchanging the \(x\)- and \(y\)-coordinates.
    Using the The Midpoint Formula, we can exploit this relationship to describe symmetry between the graph of \(f\) and the graph of \(f^{-1}\) as follows. Each \(a \in A\) corresponds to a point \((a,f(a))\) on the graph of \(f\) and a point \((f(a),a)\) on the graph of \(f^{-1}\text{.}\) The midpoint of the line that connects these two points has coordinates
    \begin{equation*} x = \frac{a + f(a)}{2} \quad\text{and}\quad y = \frac{f(a) + a}{2}. \end{equation*}
    Since the \(x\)- and \(y\)-coordinates are the same, the midpoint lies on the line \(y = x\text{.}\) This says that if we were to plot \(f\)and \(f^{-1}\) together, the image would have symmetry about the line \(y = x\text{.}\) This allows us to conclude the following.

    Theorem 4.4.1.

    The graph of \(f^{-1}\) is obtained by reflecting the graph of \(f\) across the line \(y = x\text{.}\)

    Subsection 4.4.2 The Horizontal Line Test

    The importance of Theorem 4.4.1 is that it provides a geometric condition for testing whether a function is invertible.

    Theorem 4.4.2.

    Assume \(A, B \subseteq \R\text{.}\) A function \(f \colon A \to B\) is invertible if and only if the graph
    \begin{equation*} \left\{(f(x), x) \;\middle\vert\; x \in A\right\} \end{equation*}
    represents a function.
    We can improve Theorem 4.4.2 significantly with a few key observations. Recall that we can test whether a graph represents a function using The Vertical Line Test. The graph of the vertical line \(x = a\text{,}\) where \(a\) is a constant, is the set of points
    \begin{equation*} \left\{(a,t) \;\middle\vert\; t \in \R\right\}. \end{equation*}
    Reflecting this graph across the line \(y = x\) interchanges the two coordinates, so we can express the resulting graph as the set
    \begin{equation*} \left\{(t,a) \;\middle\vert\; t \in \R\right\}. \end{equation*}
    As the \(y\)-coordinate is fixed and the \(x\)-coordinate is allowed to vary, we can recognize this as the horizontal line \(y = a\text{.}\) This provides a geometric condition for invertibility that depends only on the graph of the original function \(f\text{.}\)

    Theorem 4.4.3. The Horizontal Line Test.

    Assume \(A, B \subseteq \R\text{.}\) The function \(f \colon A \to B\) is invertible if and only if every horizontal line intersects the graph of \(f\) in at most one point.

    Example 4.4.4.

    Consider the graph of the function \(f \colon \R \to \R\) defined by \(f(x) = x^2\text{.}\) When we observed that for every positive number \(y\text{,}\) there are two \(x\)-values that square to \(y\text{,}\) we were observing that every horizontal line intersects the parabola in two places.
    When we restrict the domain to the set \([0,\infty)\text{,}\) we fix this problem by removing the left half of the parabola. This makes the function invertible with inverse the principal square root function.
    On the other hand, we could also remove the right-hand side of the parabola. This creates another invertible function. The inverse of this function is the reflection of the principal square root across the \(x\)-axis.

    Subsection 4.4.3 Root Functions

    A useful consequence of The Horizontal Line Test is that we may define functions with rational exponents.

    Definition 4.4.8. Odd Root Function.

    Assume \(n\) is an odd, positive integer and \(f \colon \R \to \R\) is the function defined by \(f(x) = x^n\text{.}\) The \(n^\text{th}\) root function \(\sqrt[n]{\phantom{x}} \colon \R \to \R\) given by \(\sqrt[n]{x} = x^{1/n}\) is defined to be the inverse of \(f\text{.}\)

    Definition 4.4.9. Even Root Function.

    Assume \(n\) is an even, positive integer and \(f \colon [0,\infty) \to \R\) is the function defined by \(f(x) = x^n\text{.}\) The \(n^\text{th}\) root function \(\sqrt[n]{\phantom{x}} \colon \R \to [0,\infty)\) given by \(\sqrt[n]{x} = x^{1/n}\) is defined to be the inverse of \(f\text{.}\)
