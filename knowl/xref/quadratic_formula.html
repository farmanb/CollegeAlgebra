Formula 3.8.3. The Quadratic Formula.

The quadratic equation
\begin{equation*} ax^2 + bx + c = 0 \end{equation*}
has either two, one, or zero real solutions, depending on The Discriminant, \(D = b^2 - 4ac\text{.}\)
  1. If \(D > 0\text{,}\) then the two solutions are
    \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{D}}{2a}. \end{equation*}
  2. If \(D = 0\text{,}\) then the only solution is
    \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} = \frac{-b}{2a}. \end{equation*}
  3. If \(D \lt 0\text{,}\) then there are no real solutions.
