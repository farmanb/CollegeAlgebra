Example 3.2.4.

Find the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\text{.}\)
Solution.
To find the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\text{,}\) we substitute \(0\) for \(y\) to get
\begin{align*} 0 \amp= -\frac{2}{3}x + 2 \amp\amp \text{Add}\ -\frac{2}{3}x\ \text{to both sides}\\ \frac{2}{3}x \amp = 2 \amp\amp \text{Multiply both sides by}\ \frac{3}{2}\\ x \amp = \left(\frac{3}{2}\right) \amp\amp \text{Simplify}\\ x \amp = 3. \end{align*}
Therefore the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\) is the point \((3,0)\text{.}\)
