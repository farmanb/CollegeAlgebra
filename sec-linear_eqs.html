    Section 3.3 Linear Functions

    Objectives

    • Recognize linear functions.
    • Find \(x\)-intercepts of a linear function.
    • Find the \(y\)-intercept of a linear function.
    • Graph a linear function.
    In this section, we study linear functions.

    Definition 3.3.1. Linear Function.

    A function \(f \colon \R \to \R\) is linear if there exists a number \(m\) such that for all numbers \(a\) and \(b\)
    \begin{equation*} m = \frac{f(b) - f(a)}{b - a} \end{equation*}

    Definition 3.3.2. Standard Form of a Linear Equation.

    A linear equation in the variables \(x\) and \(y\) is an equation that can be placed in the form
    \begin{equation*} ax + by = c. \end{equation*}
    This is called the standard form of a linear equation in two variables.

    Example 3.3.3.

    Place the linear equation
    \begin{equation*} x + 3y - 2x + 7 = 5 + y - 4x \end{equation*}
    into Definition 3.3.2.
    Solution.
    The equation
    \begin{equation*} x + 3y - 2x + 7 = 5 + y - 4x \end{equation*}
    can be placed in standard form as follows. First, move the variables to the left-hand side of the equation. Subtract \(y - 4x\) from both sides
    \begin{equation*} x - 2x + 3y + 7 - (y - 4x) = 5 \end{equation*}
    and distribute the negative
    \begin{equation*} x - 2x + 3y + 7 - y + 4x = 5. \end{equation*}
    Collect like terms
    \begin{equation*} x - 2x + 4x + 3y - y + 7 = 5 \end{equation*}
    and simplify
    \begin{equation*} 3x + 2y + 7 = 5. \end{equation*}
    Finally, subtract 7 from both sides of the equation
    \begin{equation*} 3x + 4y = 5 - 7 \end{equation*}
    and simplify to arrive at the standard form
    \begin{equation*} 3x + 4y = -2. \end{equation*}

    Example 3.3.4.

    Not all equations are linear. For example, the equation \(xy + y - 7 = 2\) is not linear because of the term \(xy\text{.}\)

    Subsection 3.3.1 Graphing Linear Equations

    Recall that the Graph of an Equation of an equation consists of all points \((x,y)\) that are a Solution to an Equation to the equation. The graph of a linear equation is always a line. We can graph the equation by finding any two points on the line and drawing a line through them.

    Subsubsection 3.3.1.1 Horizontal Lines

    If \(a = 0\) and \(b \neq 0\text{,}\) then the standard form is \(by = c\text{.}\) Dividing both sides of the equation by \(b\) allows us to write
    \begin{equation*} y = \frac{c}{b}. \end{equation*}
    The solutions to this equation are points of the form \(\left(x, \frac{c}{b}\right)\text{,}\) where \(x\) is allowed to be any real number. The graph of this equation is a horizontal line.
    Example 3.3.5.
    Graph the linear equation \(y = 0\text{.}\)
    Solution.
    The solutions to the linear equation \(y = 0\) have the form \((x, 0)\text{,}\) where \(x\) is allowed to be any real number. The graph of this equation forms the \(x\)-axis in the Cartesian plane.
    Example 3.3.7.
    Graph the linear equation \(0x + 3y = 7\text{.}\)
    Solution.
    The linear equation \(0x + 3y = 7\) simplifies to \(y = \frac{7}{3}\text{.}\) The graph of this equation is a horizontal line \(\frac{7}{3}\) units above the \(x\)-axis.

    Subsubsection 3.3.1.2 Vertical Lines

    If \(a \neq 0\) and \(b = 0\text{,}\) then the standard form is \(ax = c\text{.}\) Dividing both sides by \(a\) allows us to write
    \begin{equation*} x = \frac{c}{a}. \end{equation*}
    The solutions to this equation are points of the form \(\left(\frac{c}{a}, y\right)\text{,}\) where \(y\) is allowed to be any real number. The graph of this equation is a vertical line.
    Example 3.3.9.
    Graph the linear equation \(x = 0\text{.}\)
    Solution.
    The solutions to the linear equation \(x = 0\) have the form \((0, y)\text{,}\) where \(y\) is allowed to be any real number. The graph of this equation forms the \(y\)-axis in the Cartesian plane.
    Example 3.3.11.
    Graph the linear equation \(3x + 0y = 2\text{.}\)
    Solution.
    The linear equation \(3x + 0y = 2\) reduces to \(x = \frac{2}{3}\text{.}\) The graph of this equation is a vertical line \(\frac{2}{3}\) units to the right of the \(y\)-axis.

    Subsubsection 3.3.1.3 General Lines

    When \(a \neq 0\) and \(b \neq 0\text{,}\) we can graph the equation \(ax + by = c\) by finding any two points and drawing a line through them. There are always two simple points we can find on the line.
    Definition 3.3.13.
    The \(x\)-intercept the line \(ax + by = c\) is the point where the line intersects the \(x\)-axis.
    Every point on the \(x\)-axis has the form \((x,0)\text{.}\) By substituting 0 for \(y\) in the equation \(ax + by = c\) we get the equation
    \begin{equation*} c = ax + 0y = ax. \end{equation*}
    Dividing both sides by \(a\) tells us the \(x\)-coordinate of the \(x\)-intercept is
    \begin{equation*} x = \frac{c}{a}. \end{equation*}
    Example 3.3.14.
    Find the \(x\)-intercept of the linear equation \(2x + 3y = 5\text{.}\)
    Solution.
    To find the \(x\)-intercept of the linear equation \(2x + 3y = 6\text{,}\) we substitute \(0\) for \(y\) to get
    \begin{equation*} 2x = 6. \end{equation*}
    To solve for \(x\text{,}\) we divide both sides by \(2\) to see that x = 3. Therefore the \(x\)-intercept of the linear equation \(2x + 3y = 6\) is the point \((3,0)\text{.}\)
    Formula 3.3.15.
    The \(x\)-intercept of the linear equation \(ax + by = c\) is the point
    \begin{equation*} \left(\frac{c}{a},0\right). \end{equation*}
    Definition 3.3.16.
    The \(y\)-intercept the line \(ax + by = c\) is the point where the line intersects the \(y\)-axis.
    Every point on the \(y\)-axis has the form \((0,y)\text{.}\) By substituting 0 for \(x\) in the equation \(ax + by = c\) we get the equation
    \begin{equation*} c = 0x + by = by. \end{equation*}
    Dividing both sides by \(b\) tells us the \(y\)-coordinate of the \(y\)-intercept is
    \begin{equation*} y = \frac{c}{b}. \end{equation*}
    Formula 3.3.17.
    The \(y\)-intercept of the linear equation \(ax + by = c\) is the point
    \begin{equation*} \left(0,\frac{c}{b}\right). \end{equation*}
    Example 3.3.18.
    Find the \(y\)-intercept of the linear equation \(2x + 3y = 6\text{.}\)
    Solution.
    To find the \(y\)-intercept of the linear equation \(2x + 3y = 6\text{,}\) we substitute \(0\) for \(x\) to get
    \begin{equation*} 3y = 6. \end{equation*}
    To solve for \(y\text{,}\) we divide both sides by \(3\) to see that \(y = 2\text{.}\) Therefore the \(y\)-intercept of the linear equation \(2x + 3y = 6\) is the point \((0,2)\text{.}\)
    Example 3.3.19.
    Graph the equation \(2x + 3y = 6\)
    .
    Solution.
    To graph the equation \(2x + 3y = 6\text{,}\) we simply need to plot the two intercepts and draw a straight line through them.

    Worksheet 3.3.2 Working with Linear Equations
    A4US

    Objectives

    • Recognize linear equations in two variables.
    • Find \(x\)-intercepts of a linear equation.
    • Find the \(y\)-intercept of a linear equation.
    • Graph a linear equation.

    1.

    Determine whether the equation \(2x + 3yx = 3xy + 7y - 4\) is linear. If the equation is linear, put it standard form. Otherwise, explain why the equation is not linear.

    2.

    Determine whether the equation \(2(x - 2)^2 + y = 3x\) is linear. If the equation is linear, put it standard form. Otherwise, explain why the equation is not linear.

    3.

    Find the \(x\)- and \(y\)-intercepts of the linear equation \(5x - 2y = 10\) and sketch the graph of the equation.

    4.

    Match the following equations with their graphs.
    1. \(\displaystyle y + \frac{6}{4}x - 2 = 0\)
    2. \(\displaystyle 2y - x + 12 = 2x + 16\)
    3. \(\displaystyle 3y = 2x - 6\)
    4. \(\displaystyle y + x -xy = \frac{1}{3}x - 2 - xy\)
    5. \(\displaystyle x + 2y = 1 + x + y\)
    6. \(\displaystyle y + x = -1 + y\)
