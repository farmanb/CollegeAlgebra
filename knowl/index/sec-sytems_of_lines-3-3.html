Definition 4.1.1.

A system of \(m\) linear equations in the variables \(x\) and \(y\) is a collection of \(m\) equations
\begin{equation*} \begin{array}{ccccc} a_1x \amp + \amp b_1y \amp=\amp c_1\\ a_2x \amp + \amp b_2y \amp=\amp c_2\\ \vdots \amp \amp \vdots \amp \amp\vdots\\ a_mx \amp + \amp b_my \amp=\amp c_m \end{array} \end{equation*}
A solution to the system is a point \((a,b)\) that is a solution for every equation in the system.
