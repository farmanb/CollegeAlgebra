If we have a trinomial of the form \(ax^2 + bx + c\text{,}\) then we can use the formula
\begin{equation*} (x + a)^2 = x^2 + 2ax + a^2 \end{equation*}
to complete the square as follows.

Algorithm 2.1.7. Completing the Square.

  1. Factor an \(a\) from the first two terms to get
    \begin{equation*} ax^2 + bx + c = a\left(x^2 + \frac{b}{a}x\right) + c. \end{equation*}
  2. Add and subtract \(\left(\frac{b}{2a}\right)^2\) inside the parentheses to get
    \begin{equation*} ax^2 + bx + c = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
  3. Recognize the first three terms inside the parentheses as a perfect square
    \begin{equation*} ax^2 + bx + c = a\left(\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
  4. Distribute the \(a\) in front through the sum to get
    \begin{align*} ax^2 + bx + c \amp= a\left(x + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c \end{align*}
