Algorithm 4.1.4. Completing the Square.

To complete the square with the quadratic function \(f(x) = ax^2 + bx + c\)
  1. Factor an \(a\) from the first two terms to get
    \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x\right) + c. \end{equation*}
  2. Add and subtract \(\left(\frac{b}{2a}\right)^2\) inside the parentheses to get
    \begin{equation*} f(x) = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
  3. Factor the first three terms inside the parentheses as a perfect square
    \begin{equation*} f(x) = a\left(\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
  4. Distribute the \(a\) through the sum
    \begin{equation*} f(x) = a\left(x + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c. \end{equation*}
