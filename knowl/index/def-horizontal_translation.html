Definition 2.2.4. Horizontal Translation.

Let \(c\) be a positive number. We say the function \(g(x)\) is a horizontal translation of the function \(f(x)\) if \(g(x) = f(x \pm c)\text{.}\)
  • The graph of \(g(x) = f(x - c)\) is obtained by shifting the graph of \(y = f(x)\) to the right along the \(x\)-axis by \(c\) units.
  • The graph of \(g(x) = f(x + c)\) is obtained by shifting the graph of \(y = f(x)\) to the left along the \(x\)-axis by \(c\)-units.
