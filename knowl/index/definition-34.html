Definition 2.2.4.

Assume \(A \subseteq \R\text{.}\) The function \(f \colon A \to \R\) has a local maximum at \((c, f(c))\) if there exist values \(a,b \in A\) such that
  1. \(f\) is increasing on the interval \([a,c]\text{,}\) and
  2. \(f\) is decreasing on the interval \([c,b]\text{.}\)
We say \(f(c)\) is a local maximum value
