    Section 2.5 Linear Inequalities

    Objectives

    • Graph linear inequalities.
    • Graph linear absolute value inequalities.

    Definition 2.5.1. Linear Inequality.

    A linear inequality in the variables \(x\) and \(y\) is an expression that can be placed in one of the following forms
    • \(ax + by \lt c\text{,}\)
    • \(ax + by \leq c\text{,}\)
    • \(ax + by \gt c\text{,}\) or
    • \(ax + by \geq c\text{.}\)

    Definition 2.5.2. Solution to an Inequality.

    A solution to an inequality in the variables \(x\) and \(y\) is an ordered pair \((a,b)\) that make the equation true when
    1. the first coordinate, \(a\text{,}\) is substituted into the inequality for \(x\text{,}\) and
    2. the second coordinate, \(b\text{,}\) is substituted into the inequality for \(y\text{.}\)

    Definition 2.5.3. Graph of an Inequality.

    The graph of an inequality in the variables \(x\) and \(y\) consists of all the points in the plane that are solutions to the inequality.

    Example 2.5.4.

    Graph the linear inequality \(2x + 3y \lt 5\)
    Solution.
    The graph of \(2x + 3y \lt 5\) consists of all the points that are below the line \(2x + 3y = 5\text{.}\)
    We use a dashed line to indicate the points on the line \(2x + 3y = 5\) are not a solution to the inequality and shade the portion of the plane beneath the line to indicate those points are solutions to the inequality.

    Example 2.5.6.

    Graph the linear inequality \(2x + 3y \leq 5\)
    Solution.
    The graph of \(2x + 3y \leq 5\) consists of all the points that are on or below the line \(2x + 3y = 5\text{.}\)
    We use a solid line to indicate the points on the line \(2x + 3y = 5\) are a solution to the linear inequality and shade the portion of the plane beneath the line to indicate that those points are also solutions.

    Example 2.5.8.

    Graph the linear inequality \(2x + 3y \gt -5\)
    Solution.
    The graph of \(2x + 3y \gt -5\) consists of all the points that are above the line \(2x + 3y = -5\text{.}\)
    We use a dashed line to indicate the points on the line \(2x + 3y = -5\) are not solutions to the inequality and shade the portion of the plane above the line to indicate that those points are solutions.

    Example 2.5.10.

    Graph the linear inequality \(2x + 3y \geq -5\)
    Solution.
    The graph of \(2x + 3y \lt -5\) consists of all the points that are either on or above the line \(2x + 3y = -5\text{.}\)
    We use a solid line to indicate the points on the line \(2x + 3y = -5\) are solutions to the linear inequality and shade the portion of the plane above the line to indicate that those points are also solutions.

    Subsection 2.5.1 Absolute Value and Inequality

    Definition 2.5.12. Linear Absolute Value Inequality.

    A linear absolute value inequality in the variables \(x\) and \(y\) is an expression that can be placed in one of the following forms
    • \(\abs{ax + by + c} \lt d\text{,}\)
    • \(\abs{ax + by + c} \leq d\text{,}\)
    • \(\abs{ax + by + c} \gt d\text{,}\) or
    • \(\abs{ax + by + c} \geq d\text{.}\)
    Recall from Definition 1.3.1, for any number \(d\text{,}\)
    \begin{equation*} \abs{d} = \abs{d - 0} \end{equation*}
    measures the distance between \(d\) and the origin on the number line. We can see from
    \begin{equation*} \abs{-d} = \abs{(-d)^2} = \abs{d^2} = \abs{d} \end{equation*}
    that \(d\) and \(-d\) are the same distande from the origin.
    This observation tells us for an linear absolute value inequality of the form \(\abs{ax + by + c} \lt d\text{,}\) we are looking for the points \((x,y)\) in the plane such that the number \(ax + by + c\) is less than \(d\) units away from 0 on the number line. From Figure 2.5.13 we can see these are the solutions to both
    \begin{equation*} -d \lt ax + by + c \quad\text{and}\quad ax + by + c \lt d. \end{equation*}
    If we rewrite these as the linear inequalities
    \begin{equation*} -d - c \lt ax + by + c \quad\text{and}\quad ax + by \lt -c + d, \end{equation*}
    then we can see the solutions to \(\abs{ax + by + c} \lt d\) are the points that are above the line \(-d - c = ax + by + c\) and below the line \(ax + by = -c + d\text{.}\) Similarly, the solutions to \(\abs{ax + by + c} \leq d\) are the points that are either on or between the lines \(-d - c = ax + by + c\) and below the line \(ax + by = -c + d\text{.}\)
    For an linear absolute value inequality of the form \(\abs{ax + by + c} > d\text{,}\) the observation above tells us we are looking for the points \((x,y)\) in the plane such that the number \(ax + by + c\) is more than \(d\) units away from 0 on the number line. From Figure 2.5.13 we can see these are solutions to either
    \begin{equation*} ax + by +c \lt -d \quad\text{or}\quad d \lt ax + by + c. \end{equation*}
    If we rewrite these as the liner inequalities
    \begin{equation*} ax + by \lt -c - d \quad\text{or}\quad -c + d \lt ax + by, \end{equation*}
    then we can see the solutions to \(\abs{ax + by + c} \gt d\) are the points that are below the line \(ax + by = -c - d\) or above the line \(ax + by = -c + d\text{.}\) Similarly, the solutions to the linear absolute value inequality \(\abs{ax + by + c} \geq d\) are the points that are either on or below the line \(ax + by = -c - d\) or on or above the line \(ax + by = -c + d\text{.}\)

    Example 2.5.14.

    Graph the linear absolute value inequality \(\abs{2x + 3y} \lt 5\text{.}\)
    Solution.
    The solutions to the inequality \(\abs{2x + 3y} \lt 5\) are the points that are solutions to both of the linear inequalities
    \begin{equation*} -5 \lt 2x + 3y \quad\text{and}\quad 2x + 3y \lt 5. \end{equation*}
    This means the graph is the overlap between Figure 2.5.5 and Figure 2.5.9. That is, the points that are between the lines
    \begin{equation*} 2x + 3y = -5 \quad\text{and}\quad 2x + 3y = 5. \end{equation*}

    Example 2.5.16.

    Graph the linear absolute value inequality \(\abs{2x + 3y} \leq 5\text{.}\)
    Solution.
    The solutions to the inequality \(\abs{2x + 3y} \leq 5\) are the points that are solutions to both of the linear inequalities
    \begin{equation*} -5 \leq 2x + 3y \quad\text{and}\quad 2x + 3y \leq 5. \end{equation*}
    This means the graph is the overlap between Figure 2.5.7 and Figure 2.5.11. That is, the points that are either on or between the lines
    \begin{equation*} 2x + 3y = -5 \quad\text{and}\quad 2x + 3y = 5. \end{equation*}

    Example 2.5.18.

    Graph the linear absolute value inequality \(\abs{2x + 3y} \gt 5\text{.}\)
    Solution.
    The solutions to the inequality \(\abs{2x + 3y} \lt 5\) are the points that are solutions to either
    \begin{equation*} -5 \lt 2x + 3y \quad\text{or}\quad 2x + 3y \lt 5. \end{equation*}
    This means the graph is the combination of the graphs for the linear inequalities
    \begin{equation*} 2x + 3 \lt -5 \quad\text{and}\quad 2x + 3y \gt 5. \end{equation*}
    That is, the points that are either above the line \(2x + 3y = 5\) or below the line \(2x + 3y = -5\text{.}\)

    Example 2.5.20.

    Graph the linear absolute value inequality \(\abs{2x + 3y} \leq 5\text{.}\)
    Solution.
    The solutions to the inequality \(\abs{2x + 3y} \geq 5\) are the points that are solutions to either the linear inequality \(-5 \leq 2x + 3y\) or the linear inequality \(2x + 3y \leq 5.\) This means the graph is the combination of the graphs for the linear inequalities
    \begin{equation*} 2x + 3 \leq -5 \quad\text{and}\quad 2x + 3y \geq 5. \end{equation*}
    That is, the points that are either on or above the line \(2x + 3y = 5\text{,}\) or on or below the line \(2x + 3y = -5\text{.}\)
