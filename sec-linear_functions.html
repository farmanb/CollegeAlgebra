    Section 3.2 Linear Functions

    Objectives

    • Recognize linear functions.
    • Find \(x\)-intercepts of a linear function.
    • Find the \(y\)-intercept of a linear function.
    • Graph a linear function.
    In this section, we study a special kind of polynomial known as a linear function. As we will see, the name linear comes from the fact that the graph of such a function is always a line.

    Subsection 3.2.1 Linear Functions as Degree One Polynomials

    Definition 3.2.1. Linear Function.

    A function \(f \colon \R \to \R\) is linear if the The Average Rate of Change of \(f\) is constant. That is, there exists a number \(m\) such that for all numbers \(a\) and \(b\)
    \begin{equation*} m = \frac{f(b) - f(a)}{b - a}. \end{equation*}
    We call the number \(m\) the slope of \(f\text{.}\) The number \(b\) is the \(y\)-coordinate of the \(y\)-intercept \(\left(0, b\right)\text{.}\)
    In order to find the equation of a linear function, it is enough to know the slope, \(m\text{,}\) and the value of the function for a single \(x\)-value. The simplest point to find is the \(y\)-intercept: the \(x\)-coordinate is \(0\) and the \(y\)-coordinate is \(b = f(0)\text{.}\) If we write \(y = f(x)\text{,}\) then by Definition 3.2.1
    \begin{equation*} m = \frac{f(x) - f(0)}{x - 0} = \frac{y - b}{x}. \end{equation*}
    We obtain the equation for the function by solving the equation for \(y\) in terms of \(x\)
    \begin{align*} m \amp= \frac{y - b}{x} \amp\amp \text{Multiply both sides by}\ x\\ mx \amp = y - b \amp\amp \text{Add}\ b\ \text{to both sides}\\ mx + b \amp = y \amp\amp \text{Swap sides}\\ y \amp = mx + b \amp\amp \text{Replace}\ y\ \text{with}\ f(x)\\ f(x) \amp = mx + b \end{align*}

    Theorem 3.2.2. The Slope-Intercept Form of a Line.

    The slope-intercept form of a linear function is
    \begin{equation*} f(x) = mx + b \quad\text{or}\quad y = mx + b, \end{equation*}
    where \(m\) is the slope or average rate of change of \(f\text{.}\)

    Subsection 3.2.2 Graphing Linear Functions

    Consider a linear function \(y = mx + b\) with \(m \neq 0\text{.}\) Recall that the Graph of an Equation of an equation consists of all points \((x,y)\) that are a Solution to an Equation to the equation. The graph of a linear equation is always a line. We can graph the equation by finding any two points on the line and drawing a line through them. In general, the two points that are simplest to find are the two intercepts.
    We find the \(x\)-intercept by substituting \(0\) for \(y\) and solving the resulting equation for \(x\)
    \begin{align*} 0 \amp= mx + b \amp\amp \text{Subtract}\ b\ \text{from both sides}\\ mx \amp= -b \amp\amp \text{Divide both sides by}\ m\\ x \amp = -\frac{b}{m}. \end{align*}

    Theorem 3.2.3.

    The \(x\)-intercept of the linear function \(y = mx + b\) with \(m \neq 0\) is the point
    \begin{equation*} \left(-\frac{b}{m}, 0\right). \end{equation*}

    Example 3.2.4.

    Find the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\text{.}\)
    Solution.
    To find the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\text{,}\) we substitute \(0\) for \(y\) to get
    \begin{align*} 0 \amp= -\frac{2}{3}x + 2 \amp\amp \text{Add}\ -\frac{2}{3}x\ \text{to both sides}\\ \frac{2}{3}x \amp = 2 \amp\amp \text{Multiply both sides by}\ \frac{3}{2}\\ x \amp = \left(\frac{3}{2}\right) \amp\amp \text{Simplify}\\ x \amp = 3. \end{align*}
    Therefore the \(x\)-intercept of the linear equation \(y = -\frac{2}{3}x + 2\) is the point \((3,0)\text{.}\)
    We find the \(y\)-intercept by substituting \(0\) for \(x\) and solving the resulting equation for \(y\)
    \begin{align*} y \amp = m(0) + b \amp\amp \text{Simplify}\\ y \amp = b. \end{align*}

    Example 3.2.5.

    Graph the equation \(y = -\frac{2}{3}x + 2\)
    .
    Solution.
    In Example 3.2.4 we found that \((3,0)\) is the \(x\)-intercept for the function \(y = -\frac{2}{3}x + 2\text{.}\) The \(y\)-intercept for the function \(y = -\frac{2}{3}x + 2\) is the point \((0, 2)\text{.}\) To graph the equation \(y = -\frac{2}{3}x + 2\text{,}\) we simply need to plot these two points and draw a straight line through them.

    Worksheet 3.2.3 Working with Linear Functions
    A4US

    Objectives

    • Recognize linear functions.
    • Find \(x\)-intercept of a linear function.
    • Find the \(y\)-intercept of a linear function.
    • Graph a linear function.

    1.

    Determine whether the equation \(2x + 3yx = 3xy + 7y - 4\) represents a linear function of \(y\) in terms of \(x\text{.}\) If so, express the equation in the form \(y = mx + b\text{.}\) Otherwise, explain why the equation does not represent a linear function.

    2.

    Determine whether the equation \(2(x - 2)^2 + y = 3x\) represents a linear function of \(y\) in terms of \(x\text{.}\) If so, express the equation in the form \(y = mx + b\text{.}\) Otherwise, explain why the equation does not represent a linear function.

    3.

    Find the \(x\)- and \(y\)-intercepts of the linear equation \(5x - 2y = 10\) and sketch the graph of the equation.

    4.

    Match the following equations with their graphs.
    1. \(\displaystyle y + \frac{6}{4}x - 2 = 0\)
    2. \(\displaystyle 2y - x + 12 = 2x + 16\)
    3. \(\displaystyle 3y = 2x - 6\)
    4. \(\displaystyle y + x -xy = \frac{1}{3}x - 2 - xy\)
    5. \(\displaystyle x + 2y = 1 + x + y\)
    6. \(\displaystyle y + x = -1 + x\)
