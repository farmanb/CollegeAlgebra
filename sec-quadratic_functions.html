    Section 4.1 Quadratic Functions

    Objectives

    • find the vertex of a quadratic function,
    • find the \(x\)-intercept(s) of a quadratic function
    • find the \(y\)-intercept of a quadratic functions,
    • find the line of symmetry of a quadratic functions, and
    • graph a quadratic function.
    In this section, we study a polynomial functions of degree 2. These polynomial functions are often called quadratic function.

    Definition 4.1.1. Quadratic Function.

    A function \(f \colon \R \to \R\) is quadratic if it has the form
    \begin{equation*} f(x) = ax^2 + bx + c \quad\text{or}\quad y = ax^2 + bx + c, \end{equation*}
    where \(a\text{,}\) \(b\text{,}\) and \(c\) are numbers, and \(a \neq 0\text{.}\)

    Subsection 4.1.1 Graphing Quadratic Functions

    Just as with lines, quadratic functions are easier to understand through visualizations. While quadratics are slightly more complicated, we will see that using the methods of Section 2.4, we can understand the graph of a general quadratic function \(y = ax^2 + bx + c\) by understanding the graph of \(y = x^2\).
    The graph of \(y = x^2\text{,}\) and all other quadratic functions, is a parabola. . The graph of \(y = x^2\) has an important point, \((0,0)\text{,}\) called the vertex of the parabola. In order to find the vertex of an arbitrary quadratic function, we will need to change its form.

    Definition 4.1.2. The Vertex Form of a Quadratic.

    The vertex form of a quadratic equation \(f(x) = ax^2 + bx + c\) is
    \begin{equation*} f(x) = a(x - h)^2 + k \end{equation*}
    where \((h,k)\) is the vertex of the parabola.
    From this form, we can see that the graph of the quadratic function \(y = ax^2 + bx + c\) is obtained from the graph of \(y = x^2\) by a sequence of transformations.
    1. Translate the graph of \(y = x^2\) horizontally by \(h\) units.
      • If \(0 \lt h\text{,}\) then the translation is to the right.
      • If \(h \lt 0\text{,}\) then the translation is to the left.
      This operation translates the vertex from \((0,0)\) to \((h,0)\text{.}\)
    2. Scale the graph of \((x - h)^2\) by \(\abs{a}\text{.}\)
      • If \(1 \lt a\text{,}\) then the graph is stretched.
      • If \(a \lt 1\text{,}\) then the graph is compressed.
      This operation fixes the vertex at \((h,0)\text{.}\)
    3. If \(a \lt 1\text{,}\) reflect the graph of \(\abs{a}(x - h)^2\) over the \(x\)-axis This operation fixes the vertex at \((h,0)\text{.}\)
    4. Translate the graph of \(a(x - h)^2\) vertically by \(k\) units.
      • If \(k \lt 0\text{,}\) then the translation is down.
      • If \(0 \lt k\text{,}\) then the translation is up.
      This operation translates the vertex to \((h,k)\text{.}\)

    Subsubsection 4.1.1.1 Completing the Square

    Every quadratic function can be placed into vertex form. In order to do so, we will use the method of completing the square.
    Remember that when we multiply two binomials, we must distribute. We can remember how to distribute using the mnemonic FOIL: First, Outer, Inner, Last:
    We can use this method to multiply a binomial by itself
    \begin{align*} (x + a)^2 \amp = (x + a)(x + a)\\ \amp = xx + xa + ax + aa\\ \amp = x^2 + ax + ax + a^2\\ \amp = x^2 + 2ax + a^2. \end{align*}
    We call a quadratic function of the form \(y = x^2 + 2ax + a^2\) a perfect square.
    Given a quadratic equation \(y = ax^2 + bx + c\text{,}\) we can use the formula
    \begin{equation*} (x + a)^2 = x^2 + 2ax + a^2 \end{equation*}
    to complete the square as follows.
    Algorithm 4.1.4. Completing the Square.
    To place the quadratic function \(y = ax^2 + bx + c\) into vertex form
    1. Factor an \(a\) from the first two terms to get
      \begin{equation*} y = a\left(x^2 + \frac{b}{a}x\right) + c. \end{equation*}
    2. Add and subtract \(\left(\frac{b}{2a}\right)^2\) inside the parentheses to get
      \begin{equation*} y = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    3. Factor the first three terms inside the parentheses as a perfect square
      \begin{equation*} y = a\left(\left(x + \frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2 \right) + c \end{equation*}
    4. Distribute the \(a\) through the sum
      \begin{align*} y \amp= a\left(x + \frac{b}{2a}\right)^2 - a\left(\frac{b}{2a}\right)^2 + c. \end{align*}
      The number \(h = -\frac{b}{2a}\) is the \(x\)-coordinate of the vertex and the number
      \begin{align*} k \amp= - a\left(\frac{b}{2a}\right)^2 + c\\ \amp= -a\left(\frac{b^2}{4a^2}\right) + c\\ \amp= -\frac{b^2}{4a} + \frac{4ac}{4a}\\ \amp = - \frac{b^2 - 4ac}{4a} \end{align*}
      is the \(y\)-coordinate of the vertex.
    Example 4.1.5.
    Find the vertex of the quadratic function \(y = x^2 + 4x + 5\text{.}\)
    Solution.
    When the coefficient on \(x^2\) is 1, we can skip the first step and the last step:
    \begin{align*} y \amp = x^2 + 4x + 5 \amp\amp \text{Add and subtract}\ \left(\frac{4}{2}\right)^2 = 2^2\\ \amp= x^2 + 4x + 2^2 - 2^2 + 5 \amp\amp \text{Factor}\\ \amp = (x + 2)^2 - 2^2 + 5 \amp\amp \text{Simplify}\\ \amp = (x + 2)^2 - 4 + 5 \amp\amp \text{Simplify}\\ \amp = (x + 2)^2 + 1. \end{align*}
    The vertex of the quadratic function \(y = x^2 + 4x + 5\) is \((-2, 1)\text{.}\)
    Example 4.1.6.
    Find the vertex of the quadratic function \(y = 3x^2 - 4x + 5\text{.}\)
    Solution.
    When the coefficient on \(x\) is a negative number, we can still use the formula by rewriting the subtraction as addition with a negative number.
    \begin{align*} y \amp = 3x^2 - 4x + 5 \amp\amp \text{Rewrite}\\ \amp = 3x^2 + (-4)x + 5 \amp\amp \text{Factor}\ 3\ \text{from the first two terms}\\ \amp= 3\left(x^2 + \left(-\frac{4}{3}\right)x\right) + 5 \amp\amp \text{Add and subtract}\ \left(\frac{\frac{4}{3}}{2}\right)^2 = \left(\frac{2}{3}\right)^2\\ \amp = 3\left(x^2 + \left(-\frac{4}{3}\right)x + \left(-\frac{2}{3}\right)^2 - \left(-\frac{2}{3}\right)^2\right) + 5\amp\amp \text{Factor}\\ \amp = 3\left(\left(x + \left(-\frac{2}{3}\right)\right)^2 - \frac{4}{9}\right) + 5 \amp\amp \text{Distribute}\\ \amp = 3\left(x - \frac{2}{3}\right)^2 - \frac{4}{3} + \frac{15}{3} \amp\amp \text{Simplify}\\ \amp = 3\left(x - \frac{2}{3}\right)^2 + \frac{11}{3}. \end{align*}
    The vertex of the quadratic function \(y = 3x^2 -4x + 5\) is \(\left(\frac{2}{3}, \frac{11}{3}\right).\)

    Subsubsection 4.1.1.2 The Quadratic Equation

    A point \((x,0)\) in the Cartesian plane is an \(x\)-intercept for the quadratic function \(y = ax^2 + bx + c\) if the first coordinate is a solution to the quadratic equation
    \begin{equation*} 0 = ax^2 + bx + c. \end{equation*}
    To find the solutions to a quadratic equation, we use the final form from Completing the Square to rewrite the equation as
    \begin{equation*} 0 = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} \end{equation*}
    and solve this equation for \(x\)
    \begin{align*} 0 \amp= a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2 - 4ac}{4a} \amp\amp \text{Add}\ \frac{b^2 - 4ac}{4a}\ \text{to both sides}\\ a\left(x + \frac{b}{2a}\right)^2 \amp= \frac{b^2 - 4ac}{4a} \amp\amp \text{Divide both sides by}\ a\\ \left(x + \frac{b}{2a}\right)^2 \amp= \frac{b^2 - 4ac}{4a^2} \amp\amp \text{Take the square root of both sides}\\ x + \frac{b}{2a} \amp= \pm\sqrt{\frac{b^2 - 4ac}{4a^2}} \amp\amp \text{Simplify}\\ x + \frac{b}{2a} \amp= \pm\frac{\sqrt{b^2 - 4ac}}{\sqrt{4a^2}} \amp\amp \text{Simplify}\\ x + \frac{b}{2a} \amp= \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Subtract}\ \frac{b}{2a}\ \text{from both sides}\\ x \amp= - \frac{b}{2a} + \frac{\sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify}\\ x \amp= \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \amp\amp \text{Simplify} \end{align*}
    It is important to note that these solutions are real numbers only when \(b^2 - 4ac\) is a non-negative number. This is an important invariant of a quadratic function.
    Definition 4.1.7. The Discriminant.
    The discriminant of the quadratic function \(y = ax^2 + bx + c\) is the number
    \begin{equation*} D = b^2 - 4ac \end{equation*}
    The discriminant provides both algebraic and geometric information about quadratic functions. On the algebraic side, the discriminant tells us how many solutions there are to the quadratic equation \(ax^2 + bx +c = 0\text{.}\)
    Theorem 4.1.8. The Quadratic Formula.
    The quadratic equation
    \begin{equation*} ax + bx + c = 0 \end{equation*}
    has either two, one, or zero real solutions, depending on the The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then the two solutions are
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} \quad\text{and}\quad x = \frac{-b - \sqrt{D}}{2a}. \end{equation*}
    2. If \(D = 0\text{,}\) then the only solution is
      \begin{equation*} x = \frac{-b + \sqrt{D}}{2a} = \frac{-b}{2a}. \end{equation*}
    3. If \(D \lt 0\text{,}\) then there are no real solutions.
    On the geometric side, the discriminant tells us how many \(x\)-intercepts the graph of \(y = ax^2 + bx + c\) has.
    Theorem 4.1.9. The \(x\)-intercepts of a Quadratic Function.
    The quadratic function \(y = ax^2 + bx + c\) has either two, one, or zero \(x\)-intercepts, depending on the The Discriminant, \(D = b^2 - 4ac\text{.}\)
    1. If \(D > 0\text{,}\) then there are two \(x\)-intercepts,
      \begin{equation*} \left(\frac{-b + \sqrt{D}}{2a},0\right) \quad\text{and}\quad \left(\frac{-b - \sqrt{D}}{2a},\right). \end{equation*}
    2. If \(D = 0\text{,}\) then there is one \(x\)-intercept
      \begin{equation*} \left(\frac{-b}{2a},0\right). \end{equation*}
      Note this is the vertex.
    3. If \(D \lt 0\text{,}\) then there are no \(x\)-intercepts.
