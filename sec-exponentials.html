    Section 5.1 Exponential Functions

    So far, all of the functions we have encountered are either polynomials or can be constructed from polynomial equations using algebraic operations. A transcendental function is any function that cannot be constructed in this manner. While it is certainly odd to discuss functions that cannot be constructed by algebraic means in a course on algebra, many calculus courses expect students to already be familiar with these functions. We focus only on the algebraic properties of exponential and logarithmic functions.

    Definition 5.1.1.

    Assume \(a \neq 1\) is a fixed positive real number. A function \(f \colon \R \to (0,\infty)\) of the form \(f(x) = a^x\) is called an exponential function. The number \(a\) is called the base of the exponential function.

    Subsection 5.1.1 Modeling with Exponential Functions

    Exponential functions are of interest to scientists because they are useful in modeling natural phenomena.

    Example 5.1.2.

    A biologist observes a population over a six year period. The biologist measures the size of the observed population at the beginning and end of each of the years. In an effort to understand how the population is changing, the biologist records the year-to-year change as a numeric value and as a percentage of the initial population for the year.
    Using the table, the biologist notices the following relations amongst the population sizes
    \begin{align*} 96 \amp= 64 + \frac{64}{2} = 64\left(1 + \frac{1}{2}\right) = 64\left(\frac{3}{2}\right)\\ 144 \amp= 96 + \frac{96}{2} = 96\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^2\\ 216 \amp= 144 + \frac{144}{2} = 144\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^2\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^3\\ 324 \amp= 216 + \frac{216}{2} = 216\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^3\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^4\\ 486 \amp= 324 + \frac{324}{2} = 324\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^4\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^5\\ 729 \amp= 486 + \frac{486}{2} = 486\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^5\left(\frac{3}{2}\right) = 64\left(\frac{3}{2}\right)^6. \end{align*}
    This leads the biologist to conclude that the size of the population can be measured with the exponential function
    \begin{equation*} P(t) = 64\left(\frac{3}{2}\right)^t, \end{equation*}
    where \(t\) is the amount of time in years that has elapsed. This function agrees with the observations for \(t \in \{1, 2, 3, 4, 5 ,6\}\text{,}\) and predicts the size of the population at the end of the seventh year will be
    \begin{equation*} P(7) = 64\left(\frac{3}{2}\right)^7 = 1093.5 \approx 1094. \end{equation*}

    The half-life of the radioactive isotope technetium-99m is \(6\) hours [C.1]. Assume we start with a sample of \(64\) grams of technetium-99m. The table below lists the amount at the beginning and the end of each six hour period over the course of a day.
    We can model the amount of the sample remaining as a function of the number of \(6\) hour periods that have passed using the exponential function
    \begin{equation*} f(t) = 64\left(\frac{1}{2}\right)^t. \end{equation*}
    However, it would be more convenient to model the amount of the sample as a function of the number of hours that have passed. To do this, we first recognize that \(1\) hour is \(1/6^\text{th}\) of a period. We can stretch the time scale by a factor of \(6\) to obtain the function
    \begin{equation*} A(t) = f\left(\frac{t}{6}\right) = 64\left(\frac{1}{2}\right)^{\frac{t}{6}} = 64\left(\frac{1}{\sqrt[6]{2}}\right)^t \end{equation*}
    that measures the amount of the technetium-99m sample remaining after \(t\) hours have elapsed. Using this function, we have recorded the amount remaining after each hour for the first six hour period in the table below.

    Subsection 5.1.2 Properties of Exponential Functions

    The following properties of exponential functions are simply a restatement of the already familiar properties of exponents.

    Theorem 5.1.7. Properties of Exponential Functions.

    Assume \(f \colon \R \to (0, \infty)\) is defined by \(f(x) = a^x\text{.}\)
    1. \(\displaystyle f(0) = a^0 = 1.\)
    2. For all real numbers \(x\) and \(y\text{,}\)
      \begin{equation*} f(x + y) = a^{x+y} = a^xa^y = f(x)f(y). \end{equation*}
    3. For all real numbers \(x\) and \(y\text{,}\)
      \begin{equation*} f(x - y) = a^{x-y} = \frac{a^x}{a^y} = \frac{f(x)}{f(y)}. \end{equation*}
    The percentages measured in Example 5.1.2 and Example 5.1.4 are often of interest in applications.

    Definition 5.1.8.

    Let \(f\) be a function that is defined on the interval \([a,b]\text{.}\) The relative change of \(f\) on the interval \([a,b]\) is
    \begin{equation*} \frac{f(b) - f(a)}{f(a)}. \end{equation*}
    For an exponential function \(f \colon \R \to (0,\infty)\) of the form \(f(x) = a^x\text{,}\) consider the relative change on an interval of length one. Take the left endpoint to be a number \(x\) and the right endpoint to be the number \(x + 1\text{.}\) Over this interval, the relative change is
    \begin{align*} \frac{f(x + 1) - f(x)}{f(x)} \amp= \frac{a^{x + 1} - a^x}{a^x}\\ \amp= \frac{a^x a - a^x}{a^x}\\ \amp= \frac{a^x(a - 1)}{a^x}\\ \amp= a-1. \end{align*}

    Definition 5.1.9.

    Assume the function \(f \colon \R \to (0,\infty)\) is defined by \(f(x) = a^x\text{,}\) where \(0 \lt a\) and \(a \neq 1.\) The number \(r = a - 1\) is called the growth rate if \(r > 1\) or the decay rate if \(r \lt 1\text{.}\)
    When \(r > 0\text{,}\) we say the function \(f(x) = a^x\) models exponential growth and we call the number \(a = 1 + r\) the growth factor. When \(r \lt 0\text{,}\) we say the function \(f(x) = a^x\) models exponential decay and we call the number \(a = 1 + r\) the decay factor.
    Graphs of exponential functions depend on whether they model growth or decay. The general shape of exponential functions that model growth is roughly the same. The difference between the graphs is really only detectable when viewing multiple functions together at the same time: the larger the base (or growth factor), the steeper the graph.
    The graphs of exponential functions that model decay are similar. In fact, every function that models exponential decay is the reflection of a function that models exponential growth across the \(y\)-axis. For example, the exponential decay function \(f(x) = (2/3)^x\) is the reflection of the exponential growth function \(g(x) = (3/2)^x\) because
    \begin{equation*} g(-x) = \left(\frac{3}{2}\right)^{-x} = \left(\left(\frac{3}{2}\right)^{-1}\right)^x = \left(\frac{2}{3}\right)^x = f(x). \end{equation*}
    Specifying either a growth/decay rate and an initial quantity is sufficient to determine an exponential function.

    Example 5.1.12.

    Your local bank offers a savings account with a \(6\%\) annual interest that is compounded monthly. This means the bank will add \(6\%/12 = 0.5\%\) of your current balance to your account every month. In other words, your balance grows at a rate of \(0.5\%\) per month.
    The growth factor for the exponential function that models the amount of money in the account after \(t\) months is
    \begin{equation*} a = 1 + 0.5\% = 1 + \frac{0.5}{100} = 1 + 0.005 = 1.005. \end{equation*}
    Assuming you deposit \(\$1,000\) into a savings account, the balance of the account after \(t\) months is given by the function
    \begin{equation*} B(t) = 1000(1.005)^t. \end{equation*}
    Customarily, one measures investments as a function of the number of years that have elapsed, rather than months. Since there are \(12\) months in a year, we compress the time scale by a factor of \(12\) to obtain the function
    \begin{equation*} A(t) = B(12t) = 1000(1.005)^{12t} = 1000\left(1.005^{12}\right)^t \end{equation*}
    that measures the amount of money in the account after \(t\) years.
    The table below shows the amount of money in the account for a five year period, with all figures rounded to two decimal places.
    Note that the growth factor for this function is \(a = \left(1.005\right)^{12}\text{,}\) so the growth rate
    \begin{equation*} a - 1 = \left(1.005\right)^{12} - 1 \approx 6.17\% \end{equation*}
    is slightly higher than the interest rate (\(6\%\)).
    In general, we can model compound interest using the following formula.

    Formula 5.1.14. Compound Interest Formula.

    Assume \(P\) dollars are invested in an account at an annual interest rate of \(r\%\) that is compounded \(n\) times each year. The amount of money in the account after \(t\) years is
    \begin{equation*} A(t) = P\left(1 + \frac{r}{n}\right)^{nt}. \end{equation*}
    The growth rate for this function is
    \begin{equation*} \left(1 + \frac{r}{n}\right) - 1, \end{equation*}
    which measures the increase in the balance over one year as a percentage of the previous balance.

    Subsection 5.1.3 The Natural Exponential

    At the end of Example 5.1.12, we observed that the interest rate may be lower than the actual growth rate when the interest compounds more than once per year. Consider the case where the interest rate is \(r = 100\%\text{.}\) While it is absurd to expect to find an interest rate this high, it simplifies the growth factor in the Compound Interest Formula to
    \begin{equation*} \left(1 + \frac{1}{n}\right)^n. \end{equation*}
    We get a list of numbers, one for each positive integer \(n\text{.}\)

    Definition 5.1.17. Euler’s Number.

    Euler’s number is the constant \(e \approx 2.71828182846\text{.}\)
    Intriguingly, this same behavior happens for any value of \(r\text{.}\) As \(n\) gets larger, the values of the growth factor
    \begin{equation*} \left(1 + \frac{r}{n}\right)^n \end{equation*}
    get closer and closer to the constant \(e^r\text{.}\)
    If we recall that \(n\) is the number of times interest is compounded each year, we can think about letting \(n\) become arbitrarily large as saying that interest is compounding continuously.

    Formula 5.1.18. Compound Interest Formula.

    Assume \(P\) dollars are invested in an account at an annual interest rate of \(r\%\) that is compounded continuously. The amount of money in the account after \(t\) years is
    \begin{equation*} A(t) = Pe^{rt} = P(e^r)^t. \end{equation*}
    The growth rate for this function is \(e^r - 1\text{,}\) which measures the increase in the balance over one year as a percentage of the previous balance.
