Definition 2.4.1. Vertical Translation.
Let \(c\) be a positive number. We say the function \(g(x)\) is a vertical translation of the function \(f(x)\) if \(g(x) = f(x) \pm c\text{.}\)
  • The graph of \(g(x) = f(x) + c\) is obtained by shifting the graph of \(y = f(x)\) up the \(y\)-axis by \(c\) units.
  • The graph of \(g(x) = f(x) - c\) is obtained by shifting the graph of \(y = f(x)\) down the \(y\)-axis by \(c\)-units.
