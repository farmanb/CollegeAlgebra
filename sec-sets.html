    Section 1.1 Sets

    Unlike the sciences, where knowledge is often derived from observation and experimentation — a type of reasoning called inductive — mathematical truth can only be derived from deductive reasoning. All of mathematics is derived from a list of only nine statements that are assumed to be true called axioms — the curious reader can find these statements in any resource on Zermelo-Fraenkel Set Theory 1 .
    For this reason, sets and set operations are unavoidable in a careful treatment of mathematics. However, this is not a course on the foundations of mathematics. We present here only the rudiments of a set theory that includes only the pieces absolutely necessary for the development of the material relevant to basic algebra. We begin with the intuitive definition of a set.

    Definition 1.1.1.

    A set is a collection of objects called elements or members of the set. We specify that an object \(x\) is an element of the set \(S\) by writing \(x \in S\) and say "\(x\) is an element (or member) of \(S\)"

    Subsection 1.1.1 Representations of Sets

    Sets are effectively mathematical containers for things. While there is no restriction on what kinds of things we can place in a set, we will focus on sets that contain numbers. For small collections of things, we can write down the elements explicitly.

    Definition 1.1.2. Roster Notation.

    Roster notation specifies the elements of a set as a comma separated list surrounded by curly braces.
    Most interesting sets either have more elements than we want to write down or more elements than we can write down. For large sets, we use a sort of description of the elements.

    Definition 1.1.3. Set-Builder Notation.

    Set-builder notation specifies the members of a set using a variable and condition for membership in the set.
    The condition for membership must always be a statement that is either true or false. It allows us to test whether a thing belongs in the set.

    Example 1.1.4.

    Express the set with elements \(1\text{,}\) \(2\text{,}\) \(3\text{,}\) and \(4\) in Roster Notation.
    Solution.
    \(S = \{1, 2, 3, 4\}\)

    Example 1.1.5.

    Express the set that contains all positive, even integers in Set-Builder Notation.
    Solution.
    By definition, the even positive integers are all the multiples of \(2\text{:}\)
    \begin{equation*} 2 = 2 \times 1,\, 4 = 2 \times 2,\, 6 = 2 \times 3,\, 8 = 2 \times 4,\, \ldots \end{equation*}
    The ellipsis (\(\ldots\)) here means the list continues according to the pattern. Therefore we can express the set of even positive integers as
    \begin{equation*} E = \left\{2n \;\middle\vert\; n\ \text{is a positive integer}\right\}. \end{equation*}
    Note that we can use the conditions in the set-builder notation of \(E\) to test whether any thing is an element of \(E\text{.}\) For example, we can conclude a Zebra is not an element of this set because a Zebra is not a number (let alone twice an integer).
    More practically, we can test whether \(3 \in \E\) by first writing
    \begin{equation*} 3 = 1 \times 3 = \frac{2}{2} \times 3 = 2\left(\frac{3}{2}\right). \end{equation*}
    Since \(3/2 = 1.5\) is not an integer, we conclude \(3 \not \in E\text{.}\)
    Similarly, we can test whether \(1354618 \in E\) by writing
    \begin{equation*} 1354618 = 2 \times 677309. \end{equation*}
    Since \(677309\) is a positive integer, we conclude \(1354618 \in E\text{.}\)

    Subsection 1.1.2 Subsets and Equality

    In order to do work with sets effectively, we need a way to compare sets analogous to how we compare numbers. The simplest comparison we can make is sameness. For numbers, this is important because the same number can have two different representations, such as
    \begin{equation*} 4 = \frac{8}{2} \quad\text{or}\quad 7 = 3 \times 2 + 1. \end{equation*}
    The idea for sets is the same —a set might have more than one representation.

    Definition 1.1.6.

    Let \(A\) and \(B\) be sets. We say the sets \(A\) and \(B\) are equal and write \(A = B\) if \(A\) and \(B\) have the exact same elements.

    Example 1.1.7.

    The sets \(\{1,2,3\}\) and \(\{3, 1, 2\}\) are equal.

    Example 1.1.8.

    The sets \(\{1,2,3\}\) and \(\{1,2,4\}\) are not equal.
    Sets have a particularly odd quirk: they do not keep track of repetition.

    Example 1.1.9.

    The sets \(\{1,1,1,\}\) and \(\{1\}\) are equal.
    The other comparison we can make is an order relationship. For any two numbers, we indicate which one is the smaller of the two using the symbol \(\leq\text{,}\) such as
    \begin{equation*} 2 \leq 2 \quad\text{or}\quad \pi \leq 4. \end{equation*}
    For sets, we have a similar sort of ordering.

    Definition 1.1.10. Subset.

    Let \(A\) and \(B\) be sets. We say that \(A\) is a subset of \(B\) and write \(A \subseteq B\) if every element of \(A\) is also an element of \(B\text{.}\)

    Remark 1.1.11.

    Every set \(A\) is a subset of itself, \(A \subseteq A\text{.}\)

    Example 1.1.12.

    The set \(\{1,2\}\) is a subset of the set \(\{1,2,3\}\text{.}\) The set \(\{1,2,3\}\) is not a subset of the set \(\{1,2\}\text{.}\)
    A very nice property of numbers is that any two numbers \(x\) and \(y\text{,}\) one of \(x \leq y\) and \(y \leq x\) must be true. Unfortunately, this is not the case for sets. It may be the case that two sets \(A\) and \(B\) satisfy \(A \not \subseteq B\) and \(B \not \subseteq A\text{.}\) We call such sets incomparable

    Example 1.1.13.

    The set \(\{1,2,3\}\) is not a subset of \(\{4,5\}\) and the set \(\{4,5\}\) is not a subset of the set \(\{1,2,3\}\text{.}\)
    Finally, we note that in some instances we want to exclude the possibility that two objects are the same in an order relationship. With numbers, we use the \(\lt\) symbol to indicate that one number is strictly smaller than the other, such as
    \begin{equation*} 1 \lt 2 \quad\text{or}\quad 7 \lt 1125. \end{equation*}
    There is a strongly analogous notational change if we want to emphasize that the two sets are not the same.

    Definition 1.1.14. Proper Subset.

    Let \(A\) and \(B\) be sets. We say that \(A\) is a proper subset of \(B\) and write \(A \subset B\) if every element of \(A\) is also an element of \(B\) and \(A \neq B\text{.}\)

    Example 1.1.15.

    The set \(\{1,2\}\) is a proper subset of the set \(\{1,2,3\}\text{.}\) We could write either \(\{1,2\} \subseteq \{1,2,3\}\) or \(\{1,2\} \subset \{1,2,3\}\text{.}\) We use the latter when it is important to emphasize that the two sets are not the same.

    Subsection 1.1.3 Special Sets

    Certain sets appear so frequently that mathematicians use an agreed upon symbol to refer to the set. We collect a list of these sets below.

    Definition 1.1.16. The Empty Set.

    The empty set is the set with no elements, \(\emptyset\text{.}\)

    Remark 1.1.17.

    For convenience, the The Empty Set is a subset of every set.

    Definition 1.1.18. The Natural Numbers.

    The set of natural numbers is the set with elements all positive integers,
    \begin{equation*} \N = \left\{1, 2, 3, 4, \ldots\right\}. \end{equation*}

    Definition 1.1.19. The Integers.

    The set of integers is the set with elements all integers,
    \begin{equation*} \Z = \left\{\ldots, -4,-3,-2, -1, 0, 1, 2, 3, 4, \ldots\right\}. \end{equation*}

    Definition 1.1.20. The Rational Numbers.

    The set of rational numbers is the set with elements all rational numbers,
    \begin{equation*} \Q = \left\{\frac{a}{b} \;\middle\vert\; a \in \Z,\, b \in \Z,\, b \neq 0\right\}. \end{equation*}

    Definition 1.1.21. The Real Numbers.

    The set of real numbers is the set with elements all real numbers, \(\R\text{.}\)
    These sets of real numbers are related in the following way
    \begin{equation*} \N \subset \Z \subset \Q \subset \R. \end{equation*}
    Each of the containments is proper because, for example,
    • Every natural number is an integer, but \(0 \in \Z\) and \(0 \not \in \N\text{.}\)
    • Every integer \(n = n/1\) is a rational number, but \(1/2 \in \Q\) and \(1/2 \not \in \Z\text{.}\)
    • Every rational number is a real number, but \(\sqrt{2} \in \R\) and \(\sqrt{2} \not \in \Q\text{.}\)

    Subsubsection 1.1.1 Intervals

    A particularly special class of subsets of \(\R\) are the intervals. These sets arise frequently, so it is important to specify the notation and a graphic representation for each one.

    Definition 1.1.22. Open Interval.

    Let \(a\) and \(b\) be real numbers and assume that \(a \lt b\text{.}\) An open interval is a set that takes one of the following forms.
    \begin{align*} (a,b) \amp= \left\{x \in \R \;\middle\vert\; a \lt x \lt b\right\},\\ (a, \infty) \amp = \left\{x \in \R \;\middle\vert\; a \lt x \right\},\\ (-\infty, b) \amp = \left\{x \in \R \;\middle\vert\; x \lt b\right\},\, \text{or}\\ (-\infty, \infty) \amp= \R. \end{align*}

    Definition 1.1.24. Closed Interval.

    Let \(a\) and \(b\) be real numbers and assume that \(a \lt b\text{.}\) A closed interval is a set that takes one of the following forms
    \begin{align*} [a,b] \amp= \left\{x \in \R \;\middle\vert\; a \leq x \leq b\right\},\\ [a, \infty) \amp = \left\{x \in \R ;\middle\vert\; a \leq x\right\},\, \text{or}\\ (-\infty, b] \amp = \left\{x \in \R \;\middle\vert\; x \leq b\right\}.\\ (-\infty, \infty) \amp = \R. \end{align*}

    Definition 1.1.26. Half-Open Interval.

    Let \(a\) and \(b\) be real numbers and assume that \(a \lt b\text{.}\) A half-open interval from \(a\) to \(b\) takes one of the following forms.
    \begin{align*} (a,b] \amp= \left\{x \in \R \;\middle\vert\; a \lt x \leq b\right\},\\ [a,b) \amp= \left\{x \in \R \;\middle\vert\; a \leq x \lt b\right\}, \end{align*}

    Subsection 1.1.4 Basic Set Operations

    There are two common operations that we will frequently need to perform on sets.

    Definition 1.1.28. Intersection.

    Let \(A\) and \(B\) be sets. The intersection of \(A\) and \(B\) is the set
    \begin{equation*} A \cap B = \left\{x \;\middle\vert\; x \in A\ \text{and}\ x \in B\right\}. \end{equation*}

    Remark 1.1.29.

    The intersection is the largest set that is a subset of \(A\) and a subset of \(B\text{.}\)

    Example 1.1.30.

    The intersection of the intervals \([0,2)\) and \((1,3]\) is the set
    \begin{align*} [1,2) \cap (1,3] \amp = \left\{x \in \R \;\middle\vert\; 1 \leq x \lt 2\ \text{and}\ 1 \lt x \leq 3\right\}\\ \amp = \left\{x \in \R\;\middle\vert\; 1 \lt x \lt 2\right\}\\ \amp = (1,2) \end{align*}

    Definition 1.1.32. Union.

    Let \(A\) and \(B\) be sets. The union of \(A\) and \(B\) is the set
    \begin{equation*} A \cup B = \left\{x \;\middle\vert\; x \in A\ \text{or}\ x \in B\right\}. \end{equation*}

    Remark 1.1.33.

    In mathematics, the word or is inclusive. This means the condition \(x \in A\) or \(x \in B\) allows for the possibility that \(x \in A\) and \(x \in B\text{.}\)

    Remark 1.1.34.

    The union is the smallest set that contains both \(A\) and \(B\) as subsets.

    Example 1.1.35.

    The union of the two intervals \([-1,2)\) and \((1,3]\) is the set
    \begin{align*} [0,2) \cup (1,3] \amp = \left\{x \in \R \;\middle\vert\; 0 \leq x \lt 2\ \text{or}\ 1 \lt x \leq 3\right\}\\ \amp = 0 \leq x \leq 3\\ \amp = [0,3] \end{align*}
