Definition 5.1.9.

Assume the function \(f \colon \R \to (0,\infty)\) is defined by \(f(x) = a^x\text{,}\) where \(0 \lt a\) and \(a \neq 1.\) The number \(r = a - 1\) is called the growth rate if \(r > 1\) or the decay rate if \(r \lt 1\text{.}\)
When \(r > 0\text{,}\) we say the function \(f(x) = a^x\) models exponential growth and we call the number \(a = 1 + r\) the growth factor. When \(r \lt 0\text{,}\) we say the function \(f(x) = a^x\) models exponential decay and we call the number \(a = 1 + r\) the decay factor.
