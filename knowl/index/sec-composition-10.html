Example 4.2.9.

Let \(h\) be any non-zero real number and consider the function \(f \colon [0,\infty) \to \R\) defined by \(f(x) = \sqrt{x}\text{.}\) Find a simplified expression for the function
\begin{equation*} d(x) = \frac{f(x + h) - f(x)}{h} \end{equation*}
Solution.
Write
\begin{align*} d(x) \amp= \frac{\sqrt{x + h} - \sqrt{x}}{h} \amp\amp\text{Multiply RHS by}\ \frac{\sqrt{x + h} + \sqrt{x}}{\sqrt{x + h} + \sqrt{x}}\\ \amp= \frac{\left(\sqrt{x + h} - \sqrt{x}\right)\left(\sqrt{x + h} + \sqrt{x}\right)}{h\left(\sqrt{x + h} + \sqrt{x}\right)} \amp\amp \text{Simplify}\\ \amp= \frac{x + h - x}{{h\left(\sqrt{x + h} + \sqrt{x}\right)}} \amp\amp \text{Simplify}\\ \amp= \frac{h}{h\left(\sqrt{x + h} + \sqrt{x}\right)} \amp\amp\text{Cancel}\\ \amp= \frac{1}{\sqrt{x + h} + \sqrt{x}}. \end{align*}
The trick in the first step is often called rationalizing the numerator. For positive real numbers \(a\) and \(b\text{,}\) the quantity \(\sqrt{a} + \sqrt{b}\) is called the conjugate of \(\sqrt{a} - \sqrt{b}\text{,}\) and vise versa. Rationalizing is a trick that often comes in handy in calculus.
