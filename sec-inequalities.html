    Section 3.9 Polynomial Inequalities

    Similar to Section3.6, we now turn to polynomial inequalities, which are relations of the form
    • \(p(x) \lt 0\text{,}\)
    • \(p(x) \leq 0\text{,}\)
    • \(p(x) > 0\text{,}\) or
    • \(\displaystyle p(x) \geq 0.\)
    where \(p \colon \R \to \R\) is a polynomial function.

    Subsection 3.9.1 Solutions to Inequalities

    First we start with what it means to be a solution to an inequality.

    Definition 3.9.1. Solution to a Polynomial Inequality.

    A solution to an inequality in the variable \(x\) is a number \(a\) that makes the expression true when \(a\) is substituted for \(x\text{.}\)
    We can interpret this definition graphically as saying
    • The solutions to \(p(x) \lt 0\) are the \(x\)-values where the graph of \(y = p(x)\) is below the \(x\)-axis
    • The solutions to \(p(x) \leq 0\) are the \(x\)-values where the graph of \(y = p(x)\) is on or below the \(x\)-axis
    • The solutions to \(p(x) > 0\) are the \(x\)-values where the graph of \(y = p(x)\) is above the \(x\)-axis
    • The solutions to \(p(x) \geq \) are the \(x\)-values where the graph of \(y = p(x)\) is on or above the \(x\)-axis
    This tells us precisely how to solve a polynomial inequality using the roots.

    Subsection 3.9.2 Solving Polynomial Inequalities Algebraically

    The first method is purely algebraic. It is useful when you are unable to sketch the graph of the polynomial.

    Algorithm 3.9.2. Solving Polynomial Inequalities Algebraically.

    To find the solutions to a polynomial inequality involving the polynomial \(p(x)\)
    1. Find the distinct real roots of \(p\) and write them in order from smallest to largest
      \begin{equation*} r_1 \lt r_2 \lt r_3 \lt \ldots \lt r_n. \end{equation*}
    2. Find numbers \(x_1, x_2, x_3, \cdots, x_n, x_{n+1}\) such that
      \begin{equation*} x_1 \lt r_1 \lt x_2 \lt r_2 \lt x_3 \lt r_3 \lt \cdots \lt x_n \lt r_n \lt x_{n+1} \end{equation*}
    3. Evaluate \(p\) at each \(x_i\text{.}\)
      • If the inequality is satisfied for \(x_1\text{,}\) then the inequality holds on the interval \((-\infty, r_1)\text{.}\) Otherwise, the inequality does not hold on the interval \((-\infty, r_1)\text{.}\)
      • If the inequality is satisfied for \(x_i\) where \(2 \leq i \leq n\text{,}\) then the inequality holds on the interval \((r_{i-1}, r_i)\text{.}\) Otherwise, the inequality does not hold on the interval \((r_{i-1}, r_i)\text{.}\)
      • If the inequality is satisfied for \(x_{n+1}\text{,}\) then the inequality holds on the interval \((r_n,\infty)\text{.}\) Otherwise, the inequality does not hold on the interval \((r_n,\infty)\text{.}\)
    4. Write down each interval where the inequality holds. Include the endpoints if the inequality is not strict i.e. \(p(x) \geq 0\) or \(p(x) \leq 0\text{.}\) Otherwise, exclude the endpoints.

    Example 3.9.3.

    Find all solutions to the inequality
    \begin{equation*} (x + 4)(x + 2)x(x-2)(x-5) \leq 0. \end{equation*}
    Solution.
    The roots of the polynomial \(p(x) = (x + 4)(x + 2)x(x-2)(x-5)\) in order are
    \begin{equation*} -4 \lt -2 \lt 0 \lt 2 \lt 5. \end{equation*}
    We choose the test points \(-5\text{,}\) \(-3\text{,}\) \(-1\text{,}\) \(1\text{,}\) \(3\text{,}\) and \(6\text{.}\)
    • When \(x = -5\text{,}\)
      \begin{align*} p(-5) \amp= (-5 + 4)(-5 + 2)(-5)(-5 - 2)(-5 - 5)\\ \amp= (-1)(-3)(-5)(-7)(-10)\\ \amp= -(3)(5)(7)(10)\\ \amp\lt 0. \end{align*}
      This tells us the inequality is satisfied on \((-\infty, -4)\text{.}\)
    • When \(x = -3\text{,}\)
      \begin{align*} p(-3) \amp = (-3 + 4)(-3 + 2)(-3)(-3 - 2)(-3 - 5)\\ \amp= (1)(-1)(-3)(-5)(-8)\\ \amp= (3)(5)(8)\\ \amp > 0. \end{align*}
      This tells us the inequality is not satisfied on \((-4,-2)\text{.}\)
    • When \(x = -1\text{,}\)
      \begin{align*} p(-1) \amp= (-1 + 4)(-1 + 2)(-1)(-1 - 2)(-1 - 5)\\ \amp= (3)(1)(-1)(-3)(-6)\\ \amp= -(3)(3)(6)\\ \amp\lt 0. \end{align*}
      This tells us the inequality is satisfied on \((-2,0)\text{.}\)
    • When \(x = 1\text{,}\)
      \begin{align*} p(1) \amp= (1 + 4)(1 + 2)(1)(1 - 2)(1 - 5)\\ \amp= (5)(3)(1)(-1)(-4)\\ \amp= 5(3)(4)\\ \amp > 0. \end{align*}
      This tells us the inequality is not satisfied on \((0, 2)\text{.}\)
    • When \(x = 3\text{,}\)
      \begin{align*} p(3) \amp= (3 + 4)(3 + 2)(3)(3 - 2)(3 - 5)\\ \amp= (7)(5)(3)(1)(-2)\\ \amp= -(7)(5)(3)(1)\\ \amp \lt 0. \end{align*}
      This tells us the inequality is satisfied on \((2,5)\text{.}\)
    • When \(x = 6\text{,}\)
      \begin{align*} p(6) \amp= (6 + 4)(6 + 2)(6)(6 - 2)(6 - 5)\\ \amp= (10)(8)(6)(4)1)\\ \amp > 0. \end{align*}
      This tells us the inequality is not satisfied on \((5,\infty)\text{.}\)
    Therefore the inequality holds for all \(x\) in the set
    \begin{equation*} (-\infty, -4] \cup [-2,0] \cup [2,5] \end{equation*}

    Subsection 3.9.3 Solving Polynomial Inequalities Graphically

    Solving polynomial inequalities graphically is significantly simpler. However, it has the disadvantage that it requires you to be able to sketch the graph of the polynomial function.

    Example 3.9.4.

    Find all the values of \(x\) that satisfy the inequality
    \begin{equation*} x^2 \lt 6 - x. \end{equation*}
    Solution.
    First, we place the inequality into the form \(p(x) \lt 0\)
    \begin{align*} \amp x^2 \lt 6 - x \amp\amp \text{Add}\ x - 6\ \text{to both sides}\\ \amp x^2 + x - 6 \lt 0 \amp\amp\text{Factor the LHS}\\ \amp (x + 3)(x - 2) \lt 0 \end{align*}
    Since \(p(x) = x^2 + x - 6\) is a concave up parabola with roots \(x = -3\) and \(x = 2\text{,}\) we can see the inequality is satisfied for all \(x \in (-3, 2)\) from the graph of \(p(x)\)
