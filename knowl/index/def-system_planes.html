Definition 6.2.3.

A system of \(m\) linear equations in the variables \(x\text{,}\) \(y\text{,}\) and \(z\) is a collection of \(m\) equations
\begin{equation*} \begin{array}{ccccccc} a_1x \amp + \amp b_1y \amp + \amp c_1z \amp=\amp d_1\\ a_2x \amp + \amp b_2y \amp + \amp c_2z \amp=\amp d_2\\ \vdots \amp \amp \vdots \amp \amp \vdots \amp \amp\vdots\\ a_mx \amp + \amp b_my \amp + \amp c_3z \amp=\amp d_m \end{array} \end{equation*}
A solution to the system is a point \((a,b,c)\) that is a solution for every equation in the system.
